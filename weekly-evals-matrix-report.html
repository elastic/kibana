05 Concerns & Action Items

L4 Gap (All Suites)

toolSelection and schemaCompliance evaluators are absent from all 12 suites. No tool-use quality data is available at the L4 tier.

L1 Skill Invoked = 0

alerts-rag, pci-compliance, and agent-builder have the Skill Invoked evaluator wired but skills didn't load in traces. Likely trace ingestion failures from Kibana crashes during the run window.

Low MITRE Accuracy (0.32) — security-ai-rules

Rule creation struggles with correct ATT&CK technique mapping. Syntax and schema validity pass, but semantic technique selection is unreliable.

Low Factuality

alerts-rag scores 0.12 and agent-builder scores 0.09 on Factuality — responses contain claims not grounded in retrieved context despite passing primary quality thresholds.

ES|QL Functional Equivalence Near Zero

esql-generation (0.05) and security-ai-rules (0) produce syntactically valid but functionally incorrect queries. Validity passes; equivalence does not.

Auto-migrations Translation Fidelity (0.25)

Dashboard migration quality is low across the board. Panel Count Preservation at 0; Translation Result and Translated ESQL Validity also at 0.

Token Usage Data Missing

Evaluators ran but token scores weren't exported due to Kibana crashes during score ingestion. Re-run with stable cluster to capture L3 input/output/cached token metrics.

attack-discovery Deferred

Suite requires golden cluster vault auth to restore dataset JSONL. Blocked on deployment ensure-golden-evals credential provisioning.
