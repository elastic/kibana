Generate, validate, and execute ES|QL (Elasticsearch Query Language) queries against Elasticsearch. Use this skill when the user wants to query Elasticsearch data, analyze logs, aggregate metrics, explore indices, search documents, detect spikes/anomalies/change points, or produce tabular results that can be charted. Walks the agent through detecting cluster type and ES|QL feature availability, discovering indices and field mappings (including time-series mode), and composing the simplest correct ES|QL query before executing it with the platform.core.execute_esql tool. Includes references for ES|QL syntax, search functions, time-series queries, version history, generation tips, query patterns, and Query DSL to ES|QL migration.
