# MV_SUM

The MV_SUM function converts a multivalued field into a single valued field containing the sum of all its values.

## Syntax

`MV_SUM(number)`

### Parameters

#### number

A multivalue expression containing the values to be summed.

## Examples

This query creates a row with a multivalued field `a` containing the values `[3, 5, 6]`, and then calculates the sum of these values using the `MV_SUM` function, storing the result in a new field called `sum_a`.

```esql
ROW a=[3, 5, 6]
| EVAL sum_a = MV_SUM(a)
```
