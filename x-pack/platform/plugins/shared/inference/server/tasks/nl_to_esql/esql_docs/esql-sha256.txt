# SHA256

The SHA256 command computes the SHA256 hash of the input value.

## Syntax

`SHA256(input)`

### Parameters

#### input

The value to hash.

## Examples

The following query selects all rows from the `sample_data` index where the `message` column does not equal "Connection error". For each of these rows, it computes the SHA256 hash of the `message` column and stores the result in a new column called `sha256`. Finally, it keeps only the `message` and `sha256` columns in the output.

```esql
FROM sample_data
| WHERE message != "Connection error"
| EVAL sha256 = sha256(message)
| KEEP message, sha256
```
