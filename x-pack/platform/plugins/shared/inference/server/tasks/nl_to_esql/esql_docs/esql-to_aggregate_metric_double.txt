# TO_AGGREGATE_METRIC_DOUBLE

The TO_AGGREGATE_METRIC_DOUBLE function encodes a numeric value as an aggregate_metric_double.

## Syntax

`TO_AGGREGATE_METRIC_DOUBLE(number)`

### Parameters

#### number

The input value to encode. This can be a single value, a multi-valued column, or an expression.

## Examples

Encodes the numeric value `3892095203` as an aggregate_metric_double and stores it in the `agg_metric` column.

```esql
ROW x = 3892095203
| EVAL agg_metric = TO_AGGREGATE_METRIC_DOUBLE(x)
```

Encodes the list of numeric values `[5032, 11111, 40814]` as an aggregate_metric_double and stores it in the `agg_metric` column.

```esql
ROW x = [5032, 11111, 40814]
| EVAL agg_metric = TO_AGGREGATE_METRIC_DOUBLE(x)
```
