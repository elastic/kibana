# MIN

The MIN function returns the minimum value of a specified field.

## Syntax

`MIN(field)`

### Parameters

#### field

The column for which you want to find the minimum value.

## Examples

Finds the smallest value present in the `languages` column across all employees.

```esql
FROM employees
| STATS MIN(languages)
```

Finds the lowest average salary change among all employees.

```esql
FROM employees
| STATS min_avg_salary_change = MIN(MV_AVG(salary_change))
```
