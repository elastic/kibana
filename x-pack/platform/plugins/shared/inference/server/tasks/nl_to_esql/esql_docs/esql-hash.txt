      # HASH

  The HASH function computes the hash of a given input using a specified algorithm.

  ## Syntax

  `HASH(algorithm, input)`

  ### Parameters

  #### algorithm

  The hash algorithm to be used, such as "sha-1" or "md5"

  #### input

  The value to be hashed.
