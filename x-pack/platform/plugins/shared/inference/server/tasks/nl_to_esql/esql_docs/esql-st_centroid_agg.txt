# ST_CENTROID_AGG

Calculates the spatial centroid for a field containing spatial point geometry data.

## Syntax

`ST_CENTROID_AGG(field)`

### Parameters

#### field

The column containing spatial point geometry values to aggregate.

## Examples

Calculate the centroid of all airport locations:

This query retrieves all records from the `airports` index and calculates the geographic centroid (average location) of all airport locations using the `location` field.

```esql
FROM airports
| STATS centroid=ST_CENTROID_AGG(location)
```
