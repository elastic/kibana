# TO_VERSION

Converts an input string to a version value.

## Syntax

`TO_VERSION(field)`

### Parameters

#### field

Input value to convert to a version. The input can be a single- or multi-valued column or an expression.

## Examples

This query creates a single row with a column named `v`, where the value is the result of converting the string "1.2.3" into a version value.

```esql
ROW v = TO_VERSION("1.2.3")
```
This example converts the string "1.2.3" into a version value.
