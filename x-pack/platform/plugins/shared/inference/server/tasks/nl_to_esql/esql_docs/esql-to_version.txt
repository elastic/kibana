# TO_VERSION

The TO_VERSION function converts an input string to a version value.

## Syntax

`TO_VERSION(field)`

### Parameters

#### field

Input value to be converted. The input can be a single- or multi-valued column or an expression.

## Examples

The following query creates a single row with a column named `v`, where the value is the result of converting the string "1.2.3" into a version value using the `TO_VERSION` function.

```esql
ROW v = TO_VERSION("1.2.3")
```
