# POW

The `POW` function calculates the value of a base raised to the power of an exponent.

## Syntax

`POW(base, exponent)`

### Parameters

#### `base`

Numeric expression for the base. If `null`, the function returns `null`.

#### `exponent`

Numeric expression for the exponent. If `null`, the function returns `null`.

## Examples

### Example 1: Basic usage

Calculate the result of raising 2.0 to the power of 2:

```esql
ROW base = 2.0, exponent = 2
| EVAL result = POW(base, exponent)
```

### Example 2: Fractional exponent (root calculation)

Use a fractional exponent to calculate roots. For example, an exponent of `0.5` calculates the square root of the base:

```esql
ROW base = 4, exponent = 0.5
| EVAL s = POW(base, exponent)
```

## Limitations

- If the result of the calculation overflows a double, the function will return `null`.
