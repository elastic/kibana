# TO_DENSE_VECTOR

Converts a multi-valued input of numbers or a hexadecimal string into a dense_vector.

## Syntax

`TO_DENSE_VECTOR(field)`

### Parameters

#### field

The multi-valued input of numbers or hexadecimal string to convert.

## Examples

This query takes an array of integers `[1, 2, 3]`, converts it into a dense_vector using the `to_dense_vector` function, and then keeps only the resulting vector in the output.

```esql
row ints = [1, 2, 3]
| eval vector = to_dense_vector(ints)
| keep vector
```
