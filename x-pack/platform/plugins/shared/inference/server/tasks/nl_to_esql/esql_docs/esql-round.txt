# ROUND

Rounds a number to the specified number of decimal places. By default, it rounds to 0 decimal places, returning the nearest integer. If the precision is negative, it rounds to the specified number of digits left of the decimal point.

## Syntax

`ROUND(number, decimals)`

### Parameters

#### `number`

The numeric value to round. If `null`, the function returns `null`.

#### `decimals`

The number of decimal places to round to. Defaults to 0. If `null`, the function returns `null`.

## Examples

### Example 1: Rounding height to one decimal place

```esql
FROM employees
| KEEP first_name, last_name, height
| EVAL height_ft = ROUND(height * 3.281, 1)
```

This example converts the height from meters to feet and rounds the result to one decimal place.
