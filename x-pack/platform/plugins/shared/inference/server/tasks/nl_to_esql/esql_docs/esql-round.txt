# ROUND

The ROUND function rounds a number to the specified number of decimal places. By default, it rounds to 0 decimal places, returning the nearest integer. If the precision is a negative number, it rounds to the number of digits to the left of the decimal point.

## Syntax

`ROUND(number, decimals)`

### Parameters

#### number

The numeric value to round. If `null`, the function returns `null`.

#### decimals

The number of decimal places to round to. Optional. Defaults to 0. If `null`, the function returns `null`.

## Examples

This query selects the `first_name`, `last_name`, and `height` columns from the `employees` data. It then calculates each employee's height in feet by multiplying the `height` (in meters) by 3.281, and rounds the result to one decimal place, storing it in a new column called `height_ft`.

```esql
FROM employees
| KEEP first_name, last_name, height
| EVAL height_ft = ROUND(height * 3.281, 1)
```
