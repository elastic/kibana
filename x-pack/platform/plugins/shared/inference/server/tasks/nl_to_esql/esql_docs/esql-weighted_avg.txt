# WEIGHTED_AVG

The weighted average of a numeric expression.

**Syntax**
`WEIGHTED_AVG(number, weight)`

### Parameters

#### `number`

A numeric value.

#### `weight`

A numeric weight.

**Supported types**

**Example**
```esql
FROM employees
| STATS w_avg = WEIGHTED_AVG(salary, height) BY languages
| EVAL w_avg = ROUND(w_avg)
| KEEP w_avg, languages
| SORT languages
```
