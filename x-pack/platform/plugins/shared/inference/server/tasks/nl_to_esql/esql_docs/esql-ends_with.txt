# ENDS_WITH

Returns a boolean value indicating whether a keyword string ends with a specified suffix.

## Syntax

`ENDS_WITH(str, suffix)`

### Parameters

#### str

String expression to be evaluated. If `null`, the function returns `null`.

#### suffix

String expression representing the suffix to check for. If `null`, the function returns `null`.

## Examples

The following query checks if the last name of each employee ends with the letter "d". It keeps only the `last_name` field and adds a new field `ln_E` that is `true` if the last name ends with "d", and `false` otherwise.

```esql
FROM employees
| KEEP last_name
| EVAL ln_E = ENDS_WITH(last_name, "d")
```
