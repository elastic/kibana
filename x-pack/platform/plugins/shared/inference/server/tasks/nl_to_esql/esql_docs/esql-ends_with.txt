# ENDS_WITH

Returns a boolean value indicating whether a keyword string ends with a specified suffix string.

## Syntax

`ENDS_WITH(str, suffix)`

### Parameters

#### str

The string expression to evaluate. If this value is `null`, the function returns `null`.

#### suffix

The string expression to check as the suffix. If this value is `null`, the function returns `null`.

## Examples

Check if the `last_name` column ends with the letter "d":

This query selects the `last_name` field from the `employees` data, then creates a new column `ln_E` that contains `true` if the `last_name` ends with the letter "d", and `false` otherwise.

```esql
FROM employees
| KEEP last_name
| EVAL ln_E = ENDS_WITH(last_name, "d")
```
