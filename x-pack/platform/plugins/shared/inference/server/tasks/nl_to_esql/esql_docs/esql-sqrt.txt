# SQRT

The `SQRT` function calculates the square root of a given numeric value. The result is always a double. If the input is negative, infinite, or `null`, the function returns `null`.

## Syntax

`SQRT(number)`

### Parameters

#### `number`

A numeric expression. If the value is `null`, the function returns `null`.

## Examples

### Example 1: Calculating the square root of a number

```esql
ROW d = 100.0
| EVAL s = SQRT(d)
```

This example calculates the square root of `100.0` and stores the result in the column `s`.
