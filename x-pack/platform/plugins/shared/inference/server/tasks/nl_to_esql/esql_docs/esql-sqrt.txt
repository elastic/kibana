# SQRT

The SQRT function returns the square root of a numeric value. The result is always a double. If the input is a negative number or infinity, the function returns null.

## Syntax

`SQRT(number)`

### Parameters

#### number

Numeric expression. If the value is null, the function returns null.

## Examples

Calculate the square root of 100.0:

This query creates a single row with a column `d` set to 100.0, then calculates the square root of `d` and stores the result in a new column `s`.

```esql
ROW d = 100.0
| EVAL s = SQRT(d)
```
