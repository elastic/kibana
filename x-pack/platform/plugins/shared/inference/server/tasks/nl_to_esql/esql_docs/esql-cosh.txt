# COSH

Returns the hyperbolic cosine of a number.

## Syntax

`COSH(number)`

### Parameters

#### number

Numeric expression. If `null`, the function returns `null`.

## Examples

Calculates the hyperbolic cosine of the value in column `a` (which is set to 1.8).

```esql
ROW a=1.8
| EVAL cosh=COSH(a)
```
