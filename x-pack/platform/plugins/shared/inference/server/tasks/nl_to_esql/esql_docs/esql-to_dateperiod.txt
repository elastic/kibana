# TO_DATEPERIOD

The TO_DATEPERIOD function converts an input value into a `date_period` value. The input must be a valid constant date period expression.

## Syntax

`TO_DATEPERIOD(field)`

### Parameters

#### field

The input value to convert. The input must be a valid constant date period expression.

## Examples

This query demonstrates how to work with date periods in ES|QL. It creates a row with a datetime value (`x`), then calculates two new fields: `y`, which adds three days to `x` using a date period, and `z`, which subtracts three days from `x` using the TO_DATEPERIOD function to convert the string "3 days" into a date period.

```esql
ROW x = "2024-01-01"::datetime
| EVAL y = x + "3 DAYS"::date_period, z = x - TO_DATEPERIOD("3 days");
```
