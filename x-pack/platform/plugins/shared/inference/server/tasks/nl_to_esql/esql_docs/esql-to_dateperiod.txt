# TO_DATEPERIOD

Converts an input value into a `date_period` value.

**Syntax**
`TO_DATEPERIOD(field)`

### Parameters

#### `field`

Input value. The input is a valid constant date period expression.

**Supported types**

**Example**
```esql
ROW x = "2024-01-01"::datetime
| EVAL y = x + "3 DAYS"::date_period, z = x - TO_DATEPERIOD("3 days");
```
