# DAY_NAME

Returns the name of the weekday for a given date, based on the configured Locale.

## Syntax

`DAY_NAME(date)`

### Parameters

#### date

Date expression. If `null`, the function returns `null`.

## Examples

This query creates a row with a datetime value set to "1953-09-02T00:00:00.000Z" and then calculates the name of the weekday for that date, storing it in a new column called `weekday`.

```esql
ROW dt = to_datetime("1953-09-02T00:00:00.000Z")
| EVAL weekday = DAY_NAME(dt);
```
This example returns the weekday name for the date "1953-09-02T00:00:00.000Z".
