# MAX

The MAX function returns the maximum value of a field.

## Syntax

`MAX(field)`

### Parameters

#### field

The field for which to find the maximum value.

## Examples

Finds the highest value present in the `languages` field across all employees.
```esql
FROM employees
| STATS MAX(languages)
```

Calculates the largest average salary change by first averaging multiple salary change values per employee and then finding the maximum of those averages.
```esql
FROM employees
| STATS max_avg_salary_change = MAX(MV_AVG(salary_change))
```
