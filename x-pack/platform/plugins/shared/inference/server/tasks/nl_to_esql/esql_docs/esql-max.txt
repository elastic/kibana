# MAX

The MAX function returns the maximum value from a specified field.

## Syntax

`MAX(field)`

### Parameters

#### field

The field from which to find the maximum value.

## Examples

The following query retrieves the highest value found in the `languages` field from the `employees` data:

```esql
FROM employees
| STATS MAX(languages)
```

This query calculates the maximum average salary change per employee by first computing the moving average of `salary_change` for each employee, and then finding the highest of those averages:

```esql
FROM employees
| STATS max_avg_salary_change = MAX(MV_AVG(salary_change))
```
