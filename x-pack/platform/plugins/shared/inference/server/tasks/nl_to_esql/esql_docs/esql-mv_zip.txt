# MV_ZIP

Combines the values from two multivalued fields, joining each pair of values together with a specified delimiter.

## Syntax

`MV_ZIP(string1, string2, delim)`

### Parameters

#### string1

A multivalue expression.

#### string2

A multivalue expression.

#### delim

Optional. The delimiter used to join the values. If omitted, a comma (`,`) is used as the default delimiter.

## Examples

The following query creates two multivalued fields, `a` and `b`, then combines each pair of values from these fields into a new field `c`, joining them with a hyphen (`-`). The result keeps only the fields `a`, `b`, and `c` for display.

```esql
ROW a = ["x", "y", "z"], b = ["1", "2"]
| EVAL c = mv_zip(a, b, "-")
| KEEP a, b, c
```
