# MV_ZIP

Combines the values from two multivalued fields with a delimiter that joins them together.

**Syntax**
`MV_ZIP(string1, string2, delim)`

### Parameters

#### `string1`

Multivalue expression.

#### `string2`

Multivalue expression.

#### `delim`

Delimiter. Optional; if omitted, `,` is used as a default delimiter.

**Supported types**

**Example**
```esql
ROW a = ["x", "y", "z"], b = ["1", "2"]
| EVAL c = mv_zip(a, b, "-")
| KEEP a, b, c
```
