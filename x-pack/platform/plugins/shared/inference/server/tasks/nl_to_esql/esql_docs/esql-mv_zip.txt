# MV_ZIP

The MV_ZIP function combines the values from two multivalued fields, joining them together with a specified delimiter.

## Syntax

`MV_ZIP(string1, string2, delim)`

### Parameters

#### string1

Multivalue expression.

#### string2

Multivalue expression.

#### delim

Optional. The delimiter used to join the values. If omitted, a comma (`,`) is used by default.

## Examples

Combines the elements of arrays `a` and `b` into a new array `c`, joining corresponding elements with a hyphen as the delimiter.

```esql
ROW a = ["x", "y", "z"], b = ["1", "2"]
| EVAL c = mv_zip(a, b, "-")
| KEEP a, b, c
```
