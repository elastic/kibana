# TO_LOWER

Returns a new string with all characters in the input string converted to lower case.

## Syntax

`TO_LOWER(str)`

### Parameters

#### str

String expression to convert to lower case. If the value is `null`, the function returns `null`. The input can be a single-valued or multi-valued column or expression.

## Examples

Converts the value of the `message` column to lower case and stores it in a new column called `message_lower`:

```esql
ROW message = "Some Text"
| EVAL message_lower = TO_LOWER(message)
```

Converts each string in the array `["Some", "Text"]` to lower case and returns the result:

```esql
ROW v = TO_LOWER(["Some", "Text"])
```
