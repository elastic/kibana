# FLOOR

Rounds a number down to the nearest integer.

## Syntax

`FLOOR(number)`

### Parameters

#### `number`

Numeric expression. If `null`, the function returns `null`.

## Examples

### Example 1: Rounding down a floating-point number

```esql
ROW a=1.8
| EVAL a = FLOOR(a)
```

This example rounds the value `1.8` down to the nearest integer, resulting in `1`.
