# FLOOR

Rounds a number down to the nearest integer. For `long` (including unsigned) and `integer` types, this function has no effect. For `double`, it returns the closest `double` value less than or equal to the input, similar to `Math.floor`.

## Syntax

`FLOOR(number)`

### Parameters

#### number

Numeric expression to round down. If the value is `null`, the function returns `null`.

## Examples

This query creates a row with the value 1.8 in column `a`, then applies the `FLOOR` function to round it down to 1.

```esql
ROW a=1.8
| EVAL a=FLOOR(a)
```
Rounds the value 1.8 down to 1.
