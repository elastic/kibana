# MV_MAX

Converts a multivalued expression into a single-valued column containing the maximum value.

## Syntax

`MV_MAX(field)`

### Parameters

#### `field`

A multivalued expression.

## Supported Types

This function can be used with any column type, including `keyword` columns. For `keyword` columns, it selects the last string by comparing their UTF-8 representation byte by byte.

## Examples

### Example 1

```esql
ROW a=[3, 5, 1]
| EVAL max_a = MV_MAX(a)
```

Finds the maximum value in the multivalued column `a`, which contains numeric values.

### Example 2

```esql
ROW a=["foo", "zoo", "bar"]
| EVAL max_a = MV_MAX(a)
```

Finds the maximum value in the multivalued column `a`, which contains string values. The comparison is based on the UTF-8 representation of the strings.
