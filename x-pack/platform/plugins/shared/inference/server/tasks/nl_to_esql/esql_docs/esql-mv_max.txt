# MV_MAX

The MV_MAX function converts a multivalued expression into a single-valued column containing the maximum value.

## Syntax

`MV_MAX(field)`

### Parameters

#### field

The multivalue expression to evaluate.

## Examples

Calculates the maximum value from the array `[3, 5, 1]` and stores it in the column `max_a`.
```esql
ROW a=[3, 5, 1]
| EVAL max_a = MV_MAX(a)
```

Finds the maximum string value in the array `["foo", "zoo", "bar"]` based on UTF-8 comparison and stores it in `max_a`.
```esql
ROW a=["foo", "zoo", "bar"]
| EVAL max_a = MV_MAX(a)
```
