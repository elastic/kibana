# ST_EXTENT_AGG

Calculate the spatial extent over a field with geometry type. Returns a bounding box for all values of the field.

```
stack: preview
serverless: preview
```

**Syntax**
`ST_EXTENT_AGG(field)`

### Parameters

#### `field`

**Supported types**

**Example**
```esql
FROM airports
| WHERE country == "India"
| STATS extent = ST_EXTENT_AGG(location)
```
