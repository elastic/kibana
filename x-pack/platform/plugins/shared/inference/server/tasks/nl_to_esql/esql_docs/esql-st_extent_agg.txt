# ST_EXTENT_AGG

Calculates the spatial extent over a field with geometry type and returns a bounding box for all values of the field.

## Syntax

`ST_EXTENT_AGG(field)`

### Parameters

#### `field`

The field with geometry type over which the spatial extent is calculated.

## Examples

```esql
FROM airports
| WHERE country == "India"
| STATS extent = ST_EXTENT_AGG(location)
```

This example calculates the spatial extent of the `location` field for airports in India, returning a bounding box.
