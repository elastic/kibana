# COALESCE

Returns the first argument that is not null. If all arguments are null, it returns `null`.

## Syntax

`COALESCE(first, rest)`

### Parameters

#### first

Expression to evaluate.

#### rest

Other expressions to evaluate.

## Examples

This query creates a row with two columns, `a` set to null and `b` set to "b". It then uses the `COALESCE` function to return the first non-null value between `a` and `b`, which in this case is the value of `b`.

```esql
ROW a=null, b="b"
| EVAL COALESCE(a, b)
```
This example returns the value of `b` since `a` is null.
