# COALESCE

Returns the first argument that is not null. If all arguments are null, it returns `null`.

## Syntax

`COALESCE(first, rest)`

### Parameters

#### `first`

Expression to evaluate.

#### `rest`

Other expression to evaluate.

## Examples

### Example 1

```esql
ROW a=null, b="b"
| EVAL COALESCE(a, b)
```

This example evaluates the `COALESCE` function to return the first non-null value between `a` and `b`.
