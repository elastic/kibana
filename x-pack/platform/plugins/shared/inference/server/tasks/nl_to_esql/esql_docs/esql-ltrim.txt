# LTRIM

Removes leading whitespaces from a string.

## Syntax

`LTRIM(string)`

### Parameters

#### string

String expression to process. If the value is `null`, the function returns `null`.

## Examples

The following query demonstrates how to remove leading spaces from the `message` and `color` columns, and then wrap the cleaned results in single quotes for clarity:

```esql
ROW message = "   some text  ",  color = " red "
| EVAL message = LTRIM(message)
| EVAL color = LTRIM(color)
| EVAL message = CONCAT("'", message, "'")
| EVAL color = CONCAT("'", color, "'")
```
This query creates a row with two columns, `message` and `color`, each containing strings with leading spaces. It then removes the leading spaces from both columns using the `LTRIM` function. Finally, it wraps the trimmed values in single quotes using the `CONCAT` function.
