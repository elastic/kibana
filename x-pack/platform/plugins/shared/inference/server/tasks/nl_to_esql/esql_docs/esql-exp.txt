# EXP

Returns the value of e raised to the power of the given number.

## Syntax

`EXP(number)`

### Parameters

#### number

Numeric expression to use as the exponent. If `null`, the function returns `null`.

## Examples

Calculates e raised to the power of 5.0 and stores the result in column `s`.
```esql
ROW d = 5.0
| EVAL s = EXP(d)
```
