# TO_GEOHASH

The TO_GEOHASH function converts an input value to a geohash value. Conversion is successful only if the input string matches the geohash format.

## Syntax

`TO_GEOHASH(field)`

### Parameters

#### field

Input value to be converted. The input can be a single- or multi-valued column or an expression.

## Examples

Convert a string to a geohash value:

```esql
ROW string = "u3bu"
| EVAL geohash = TO_GEOHASH(string)
```
Converts the value in the "string" column ("u3bu") into its corresponding geohash value.
