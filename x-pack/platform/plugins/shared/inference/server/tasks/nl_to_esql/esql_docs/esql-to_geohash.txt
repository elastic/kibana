# TO_GEOHASH

The TO_GEOHASH function converts an input value to a `geohash` value. A string will only be successfully converted if it matches the `geohash` format, as described for the geohash grid aggregation.

## Syntax

`TO_GEOHASH(field)`

### Parameters

#### field

Input value to convert. The input can be a single- or multi-valued column or an expression.

## Examples

Convert a string to a geohash value:

```esql
ROW string = "u3bu"
| EVAL geohash = TO_GEOHASH(string)
```
This example converts the string "u3bu" into a geohash value and stores it in the `geohash` column.
