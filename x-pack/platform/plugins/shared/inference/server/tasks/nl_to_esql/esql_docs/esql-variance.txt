# VARIANCE

Calculates the population variance of a numeric field.

## Syntax

`VARIANCE(number)`

### Parameters

#### number

The numeric field for which to calculate the population variance.

## Examples

This query retrieves all records from the `employees` dataset and calculates the population variance of the `height` field, storing the result in a new column called `var_height`.

```esql
FROM employees
| STATS var_height = VARIANCE(height)
```
Calculate the population variance of the `height` field in the `employees` dataset.
