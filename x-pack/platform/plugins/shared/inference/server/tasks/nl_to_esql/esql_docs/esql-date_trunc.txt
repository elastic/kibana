## DATE_TRUNC

The `DATE_TRUNC` function rounds down a date to the closest specified interval.

## Syntax

`DATE_TRUNC(interval, date)`

### Parameters

#### `interval`

The interval to which the date is rounded down, expressed using the timespan literal syntax.

#### `date`

The date expression to be truncated.

## Examples

### Example 1: Truncate hire dates to the year

Truncate the `hire_date` field to the beginning of the year:

```esql
FROM employees
| KEEP first_name, last_name, hire_date
| EVAL year_hired = DATE_TRUNC(1 year, hire_date)
```

### Example 2: Number of hires per year

Combine `DATE_TRUNC` with the `STATS` command to create a histogram of hires per year:

```esql
FROM employees
| EVAL year = DATE_TRUNC(1 year, hire_date)
| STATS hires = COUNT(emp_no) BY year
| SORT year
```

### Example 3: Hourly error rate

Calculate the hourly error rate by truncating timestamps to the hour and aggregating error occurrences:

```esql
FROM sample_data
| EVAL error = CASE(message LIKE "*error*", 1, 0)
| EVAL hour = DATE_TRUNC(1 hour, @timestamp)
| STATS error_rate = AVG(error) BY hour
| SORT hour
```
