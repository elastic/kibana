# LOCATE

Returns an integer indicating the position of a keyword substring within another string. If the substring is not found, the function returns `0`. String positions start from `1`.

## Syntax

`LOCATE(string, substring, start)`

### Parameters

#### string

The input string in which to search.

#### substring

The substring to locate within the input string.

#### start

The start index from which to begin the search.

## Examples

Find the position of the substring "ll" within the string "hello":

This query creates a row with the value "hello" and then calculates the position where the substring "ll" first appears in that string. The result is stored in the column `a_ll`.

```esql
ROW a = "hello"
| EVAL a_ll = LOCATE(a, "ll")
```
