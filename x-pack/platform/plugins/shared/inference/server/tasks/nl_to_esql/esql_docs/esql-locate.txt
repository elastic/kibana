# LOCATE

Returns an integer indicating the position of a keyword substring within another string. Returns `0` if the substring is not found. String positions start from `1`.

## Syntax

`LOCATE(string, substring, start)`

### Parameters

#### string

The input string to search within.

#### substring

The substring to locate in the input string.

#### start

The start index from which to begin the search.

## Examples

Finds the position of the substring "ll" within the string "hello".

```esql
ROW a = "hello"
| EVAL a_ll = LOCATE(a, "ll")
```
