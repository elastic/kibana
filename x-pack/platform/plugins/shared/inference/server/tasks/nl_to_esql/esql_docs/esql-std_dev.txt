# STD_DEV

The population standard deviation of a numeric field.

**Syntax**
`STD_DEV(number)`

### Parameters

#### `number`

**Supported types**

**Examples**
```esql
FROM employees
| STATS std_dev_height = STD_DEV(height)
```

The expression can use inline functions. For example, to calculate the population standard deviation of each employeeâ€™s maximum salary changes, first use `MV_MAX` on each row, and then use `STD_DEV` on the result
```esql
FROM employees
| STATS stddev_salary_change = STD_DEV(MV_MAX(salary_change))
```
