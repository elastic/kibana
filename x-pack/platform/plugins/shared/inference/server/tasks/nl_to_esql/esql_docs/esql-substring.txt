# SUBSTRING

Returns a substring from a string, based on a specified start position and an optional length.

## Syntax

`SUBSTRING(string, start, length)`

### Parameters

#### string

The string expression to extract the substring from. If this value is `null`, the function returns `null`.

#### start

The position to start extracting the substring.

#### length

Optional. The number of characters to include in the substring, starting from the start position. If omitted, all characters after `start` are returned.

## Examples

Extracts the first three characters from each employee's last name.

```esql
FROM employees
| KEEP last_name
| EVAL ln_sub = SUBSTRING(last_name, 1, 3)
```

Extracts the last three characters from each employee's last name by starting three characters from the end.

```esql
FROM employees
| KEEP last_name
| EVAL ln_sub = SUBSTRING(last_name, -3, 3)
```

Extracts all characters from each employee's last name except for the first character.

```esql
FROM employees
| KEEP last_name
| EVAL ln_sub = SUBSTRING(last_name, 2)
```
