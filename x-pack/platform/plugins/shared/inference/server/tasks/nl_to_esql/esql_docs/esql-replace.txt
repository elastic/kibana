# REPLACE

The REPLACE function substitutes any match of the regular expression `regex` in the string `str` with the replacement string `newStr`.

## Syntax

`REPLACE(string, regex, newString)`

### Parameters

#### string

String expression to search for matches.

#### regex

Regular expression pattern to match within the string.

#### newString

Replacement string to substitute for each match.

## Examples

Replaces the word "World" with "Universe" in the input string.

```esql
ROW str = "Hello World"
| EVAL str = REPLACE(str, "World", "Universe")
```

Removes all spaces from the input string.

```esql
ROW str = "Hello World"
| EVAL str = REPLACE(str, "\\\\s+", "")
```
