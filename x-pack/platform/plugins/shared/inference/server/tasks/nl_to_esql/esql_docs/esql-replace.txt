# REPLACE

The REPLACE function substitutes any match of the regular expression `regex` in the string `str` with the replacement string `newStr`.

## Syntax

`REPLACE(string, regex, newString)`

### Parameters

#### string

String expression to search for matches.

#### regex

Regular expression pattern to match within the string.

#### newString

Replacement string to substitute for each match.

## Examples

Replace any occurrence of the word "World" with the word "Universe":

This query creates a row with the string "Hello World" and then replaces every occurrence of "World" with "Universe", resulting in "Hello Universe".

```esql
ROW str = "Hello World"
| EVAL str = REPLACE(str, "World", "Universe")
```

Remove all spaces from the string:

This query creates a row with the string "Hello World" and then removes all spaces by replacing any sequence of whitespace characters with an empty string, resulting in "HelloWorld".

```esql
ROW str = "Hello World"
| EVAL str = REPLACE(str, "\\\\s+", "")
```
