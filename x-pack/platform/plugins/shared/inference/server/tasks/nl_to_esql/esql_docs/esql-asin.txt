# ASIN

Returns the arcsine of the input numeric expression as an angle in radians.

## Syntax

`ASIN(number)`

### Parameters

#### number

A number between -1 and 1. If the value is `null`, the function returns `null`.

## Examples

This query creates a single row with a column `a` set to 0.9, then calculates the arcsine of `a` and stores the result in a new column called `asin`.

```esql
ROW a=.9
| EVAL asin=ASIN(a)
```
