# LAST_OVER_TIME

Calculates the latest value of a field, where recency is determined by the `@timestamp` field.

## Syntax

`LAST_OVER_TIME(field, window)`

### Parameters

#### field

The metric field to calculate the latest value for.

#### window

The time window over which to find the latest value.

## Examples

Calculates the maximum network cost value observed in each cluster and 1-minute time bucket, using the most recent network cost value within each bucket.

```esql
TS k8s
| STATS max_cost=MAX(LAST_OVER_TIME(network.cost)) BY cluster, time_bucket = TBUCKET(1minute)
```
