# TO_CARTESIANSHAPE

Converts an input value to a `cartesian_shape` value. A string will only be successfully converted if it follows the WKT (Well-Known Text) format.

## Syntax

`TO_CARTESIANSHAPE(field)`

### Parameters

#### field

The input value to convert. This can be a single- or multi-valued column, or an expression.

## Examples

Convert WKT strings to cartesian shapes:

This query creates a row containing an array of WKT (Well-Known Text) strings representing geometric shapes. It then expands the array so each WKT string becomes its own row, and finally converts each WKT string into a `cartesian_shape` value using the `TO_CARTESIANSHAPE` function.

```esql
ROW wkt = ["POINT(4297.11 -1475.53)", "POLYGON ((3339584.72 1118889.97, 4452779.63 4865942.27, 2226389.81 4865942.27, 1113194.90 2273030.92, 3339584.72 1118889.97))"]
| MV_EXPAND wkt
| EVAL geom = TO_CARTESIANSHAPE(wkt)
```
This example expands a list of WKT strings and converts each to a cartesian shape.
