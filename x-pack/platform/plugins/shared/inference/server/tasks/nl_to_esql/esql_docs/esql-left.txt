# LEFT

Returns the substring that extracts *length* chars from *string* starting from the left.

**Syntax**
`LEFT(string, length)`

### Parameters

#### `string`

The string from which to return a substring.

#### `length`

The number of characters to return.

**Supported types**

**Example**
```esql
FROM employees
| KEEP last_name
| EVAL left = LEFT(last_name, 3)
```
