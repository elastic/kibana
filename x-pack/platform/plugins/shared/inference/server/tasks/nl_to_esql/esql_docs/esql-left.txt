# LEFT

The LEFT function returns a substring containing a specified number of characters from the beginning (left side) of a string.

## Syntax

`LEFT(string, length)`

### Parameters

#### string

The string from which the substring will be extracted.

#### length

The number of characters to return from the left side of the string.

## Examples

The following query selects the `last_name` column from the `employees` data, and then creates a new column called `left` that contains only the first three characters of each `last_name`.

```esql
FROM employees
| KEEP last_name
| EVAL left = LEFT(last_name, 3)
```
