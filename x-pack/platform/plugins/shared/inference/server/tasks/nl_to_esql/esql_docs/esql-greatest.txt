# GREATEST

The GREATEST function returns the maximum value from multiple columns. When used with string fields, it returns the last string in alphabetical order. For boolean columns, it returns `true` if any value is `true`.

## Syntax

`GREATEST(first, rest)`

### Parameters

#### first

The first column to evaluate.

#### rest

The remaining columns to evaluate.

## Examples

The following query creates a row with two columns, `a` and `b`, having values 10 and 20 respectively. It then calculates the greatest value between these two columns and stores the result in a new column `g`.

```esql
ROW a = 10, b = 20
| EVAL g = GREATEST(a, b)
```
