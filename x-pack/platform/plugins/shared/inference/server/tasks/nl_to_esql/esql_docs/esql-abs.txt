# ABS

Returns the absolute value.

**Syntax**
`ABS(number)`

### Parameters

#### `number`

Numeric expression. If `null`, the function returns `null`.

**Supported types**

**Examples**
```esql
ROW number = -1.0
| EVAL abs_number = ABS(number)
```

```esql
FROM employees
| KEEP first_name, last_name, height
| EVAL abs_height = ABS(0.0 - height)
```
