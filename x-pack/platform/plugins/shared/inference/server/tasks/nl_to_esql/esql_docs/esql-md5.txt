# MD5

Computes the MD5 hash of the input (if the MD5 hash is available on the JVM).

**Syntax**
`MD5(input)`

### Parameters

#### `input`

Input to hash.

**Supported types**

**Example**
```esql
FROM sample_data
| WHERE message != "Connection error"
| EVAL md5 = md5(message)
| KEEP message, md5
```
