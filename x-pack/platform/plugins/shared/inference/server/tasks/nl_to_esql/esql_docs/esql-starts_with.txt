# STARTS_WITH

Returns a boolean value indicating whether a keyword string starts with a specified prefix string.

## Syntax

`STARTS_WITH(str, prefix)`

### Parameters

#### str

String expression to evaluate. If `null`, the function returns `null`.

#### prefix

String expression representing the prefix to check for. If `null`, the function returns `null`.

## Examples

Checks if the `last_name` column starts with the letter "B" and stores the result in the `ln_S` column.
```esql
FROM employees
| KEEP last_name
| EVAL ln_S = STARTS_WITH(last_name, "B")
```
