# STARTS_WITH

Returns a boolean that indicates whether a keyword string starts with another string.

**Syntax**
`STARTS_WITH(str, prefix)`

### Parameters

#### `str`

String expression. If `null`, the function returns `null`.

#### `prefix`

String expression. If `null`, the function returns `null`.

**Supported types**

**Example**
```esql
FROM employees
| KEEP last_name
| EVAL ln_S = STARTS_WITH(last_name, "B")
```
