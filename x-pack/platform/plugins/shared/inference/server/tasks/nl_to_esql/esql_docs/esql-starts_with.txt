# STARTS_WITH

Returns a boolean value indicating whether a keyword string starts with a specified prefix string.

## Syntax

`STARTS_WITH(str, prefix)`

### Parameters

#### str

The string expression to evaluate. If this value is `null`, the function returns `null`.

#### prefix

The prefix string to check for at the start of `str`. If this value is `null`, the function returns `null`.

## Examples

The following query checks whether each employee's last name starts with the letter "B". It keeps only the `last_name` column and adds a new column `ln_S` that contains `true` if the last name starts with "B", and `false` otherwise.

```esql
FROM employees
| KEEP last_name
| EVAL ln_S = STARTS_WITH(last_name, "B")
```
