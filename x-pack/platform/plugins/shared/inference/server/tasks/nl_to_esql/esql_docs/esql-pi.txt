# PI

The PI function returns the mathematical constant Pi, which is the ratio of a circleâ€™s circumference to its diameter.

## Syntax

`PI()`

### Parameters

This function does not take any parameters.

## Examples

The following query returns the value of Pi as a column in the result:

```esql
ROW PI()
```
This query creates a single row with a column containing the value of Pi (approximately 3.14159).
