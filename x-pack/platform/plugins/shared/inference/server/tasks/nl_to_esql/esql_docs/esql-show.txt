# SHOW

The SHOW command returns information about the deployment and its capabilities.

## Syntax

`SHOW item`

### Parameters

#### item

Can only be INFO.

## Examples

Use SHOW INFO to return the deploymentâ€™s version, build date, and hash.

This query retrieves general information about the Elasticsearch deployment, such as the version, build date, and build hash.

```esql
SHOW INFO
```
