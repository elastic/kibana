# SHOW

The SHOW command returns information about the deployment and its capabilities.

## Syntax

`SHOW item`

### Parameters

#### item

Can only be INFO.

## Examples

Use SHOW INFO to return the deploymentâ€™s version, build date, and hash.

The following query retrieves general information about the Elasticsearch deployment, including its version, build date, and build hash.

```esql
SHOW INFO
```
