# VARIANCE_OVER_TIME

Calculates the population variance of a numeric field over a specified time window.

## Syntax

`VARIANCE_OVER_TIME(field, window)`

### Parameters

#### field

The metric field to calculate the variance for.

#### window

The time window over which to compute the variance.

## Examples

The following query calculates the average variance of the `network.cost` field for each cluster, grouped by one-minute time buckets. It uses the `VARIANCE_OVER_TIME` function to compute the variance, then averages these values per cluster and minute.

```esql
TS k8s
| STATS avg_var_cost=AVG(VARIANCE_OVER_TIME(network.cost)) BY cluster, time_bucket = TBUCKET(1minute)
```
