# MV_MIN

The MV_MIN function converts a multivalued expression into a single-valued column containing the minimum value.

## Syntax

`MV_MIN(field)`

### Parameters

#### field

A multivalue expression to evaluate.

## Examples

This query creates a row with a multivalued field `a` containing the numbers 2 and 1, then calculates the minimum value in `a` and stores it in a new column `min_a`.
```esql
ROW a=[2, 1]
| EVAL min_a = MV_MIN(a)
```

This query creates a row with a multivalued field `a` containing the strings "foo" and "bar", then finds the minimum string value (based on UTF-8 byte order) and stores it in a new column `min_a`.
```esql
ROW a=["foo", "bar"]
| EVAL min_a = MV_MIN(a)
```
