# SPLIT

The `SPLIT` function divides a single-valued string into multiple strings based on a specified delimiter.

## Syntax

`SPLIT(string, delim)`

### Parameters

#### `string`

String expression. If the value is `null`, the function returns `null`.

#### `delim`

Delimiter used to split the string. Only single-byte delimiters are supported.

## Examples

### Example 1

Split a string containing multiple words separated by semicolons into individual strings:

```esql
ROW words="foo;bar;baz;qux;quux;corge"
| EVAL word = SPLIT(words, ";")
```
