# FROM_BASE64

The FROM_BASE64 command decodes a base64-encoded string.

## Syntax

`FROM_BASE64(string)`

### Parameters

#### string

A base64 string to be decoded.

## Examples

The following query creates a row with a column `a` containing the base64-encoded string `"ZWxhc3RpYw=="`. It then decodes this string using the `FROM_BASE64` function and stores the decoded result in a new column `d`.

```esql
ROW a = "ZWxhc3RpYw=="
| EVAL d = FROM_BASE64(a)
```
