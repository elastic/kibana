# FROM_BASE64

The FROM_BASE64 command decodes a base64-encoded string.

## Syntax

`FROM_BASE64(string)`

### Parameters

#### string

A base64-encoded string to decode.

## Examples

This query creates a row with a column `a` containing the base64-encoded string "ZWxhc3RpYw==", then decodes it using the FROM_BASE64 function and stores the decoded value in a new column `d`.

```esql
ROW a = "ZWxhc3RpYw=="
| EVAL d = FROM_BASE64(a)
```
Decode the base64 string "ZWxhc3RpYw==" and store the result in column `d`.
