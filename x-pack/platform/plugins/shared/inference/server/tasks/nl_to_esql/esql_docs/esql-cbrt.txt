# CBRT

The CBRT function returns the cube root of a number. The input can be any numeric value, and the return value is always a double. Cube roots of infinities return null.

## Syntax

`CBRT(number)`

### Parameters

#### number

Numeric expression. If the value is null, the function returns null.

## Examples

The following query creates a row with a single column `d` set to 1000.0, then calculates the cube root of `d` and stores the result in a new column `c`.

```esql
ROW d = 1000.0
| EVAL c = CBRT(d)
```
