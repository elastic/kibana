# CBRT

The `CBRT` function calculates the cube root of a given numeric value. The result is always returned as a double.

## Syntax

`CBRT(number)`

### Parameters

#### `number`

A numeric expression. If the input is `null`, the function returns `null`.

## Examples

```esql
ROW d = 1000.0
| EVAL c = cbrt(d)
```

Calculate the cube root of the value `1000.0`.
