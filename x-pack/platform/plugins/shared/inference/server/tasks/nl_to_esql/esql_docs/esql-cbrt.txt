# CBRT

Returns the cube root of a number. The input can be any numeric value, and the return value is always a double. If the input is infinity, the result is null.

## Syntax

`CBRT(number)`

### Parameters

#### number

Numeric expression. If the value is `null`, the function returns `null`.

## Examples

The following query creates a row with a value of 1000.0 and then calculates its cube root, storing the result in a new column named `c`:

```esql
ROW d = 1000.0
| EVAL c = CBRT(d)
```
