# CONCAT

The CONCAT function joins two or more strings together.

## Syntax

`CONCAT(string1, string2)`

### Parameters

#### string1

The first string to concatenate.

#### string2

The second string to concatenate.

## Examples

This query selects the `first_name` and `last_name` columns from the `employees` data source, then creates a new column called `fullname` by joining the `first_name`, a space character, and the `last_name` together.

```esql
FROM employees
| KEEP first_name, last_name
| EVAL fullname = CONCAT(first_name, " ", last_name)
```
This example creates a new column `fullname` by concatenating the `first_name`, a space, and the `last_name` columns.
