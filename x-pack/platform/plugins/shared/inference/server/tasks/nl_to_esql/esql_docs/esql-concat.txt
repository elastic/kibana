# CONCAT

Concatenates two or more strings.

## Syntax

`CONCAT(string1, string2)`

### Parameters

#### `string1`

Strings to concatenate.

#### `string2`

Strings to concatenate.

## Examples

```esql
FROM employees
| KEEP first_name, last_name
| EVAL fullname = CONCAT(first_name, " ", last_name)
```

This example creates a new column `fullname` by concatenating the `first_name` and `last_name` columns with a space in between.
