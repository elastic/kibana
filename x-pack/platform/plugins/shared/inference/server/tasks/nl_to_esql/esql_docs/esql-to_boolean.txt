# TO_BOOLEAN

Converts an input value to a boolean value. A string value of `true` (case-insensitive) is converted to the Boolean `true`. All other string values, including the empty string, are converted to `false`. A numerical value of `0` is converted to `false`, while any other numerical value is converted to `true`.

## Syntax

`TO_BOOLEAN(field)`

### Parameters

#### `field`

The input value to be converted. This can be a single- or multi-valued column or an expression.

## Examples

### Example 1: Converting strings to boolean values

```esql
ROW str = ["true", "TRuE", "false", "", "yes", "1"]
| EVAL bool = TO_BOOLEAN(str)
```

This example converts a list of string values to their corresponding boolean values. For instance, `"true"` and `"TRuE"` are converted to `true`, while `"false"`, `""`, `"yes"`, and `"1"` are converted to `false`.
