# TO_BOOLEAN

Converts an input value to a boolean value. A string value of `true` (case-insensitive) is converted to the Boolean `true`. All other string values, including the empty string, are converted to `false`. The numeric value `0` is converted to `false`, while any other numeric value is converted to `true`.

## Syntax

`TO_BOOLEAN(field)`

### Parameters

#### field

The input value to convert. This can be a single- or multi-valued column, or an expression.

## Examples

Convert an array of string values to their boolean equivalents:

```esql
ROW str = ["true", "TRuE", "false", "", "yes", "1"]
| EVAL bool = TO_BOOLEAN(str)
```
This query creates a row with an array of string values and then uses the `TO_BOOLEAN` function to convert each string in the array to its corresponding boolean value. For example, "true" and "TRuE" become `true`, while "false", the empty string, "yes", and "1" become `false` or `true` according to the conversion rules.
