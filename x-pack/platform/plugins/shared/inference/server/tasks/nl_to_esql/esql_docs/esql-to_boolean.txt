# TO_BOOLEAN

Converts an input value to a boolean value. A string value of `true` (case-insensitive) is converted to the Boolean `true`. All other string values, including the empty string, return `false`. The numerical value `0` is converted to `false`, while any other number is converted to `true`.

## Syntax

`TO_BOOLEAN(field)`

### Parameters

#### field

The input value to convert. This can be a single- or multi-valued column, or an expression.

## Examples

This query creates a row with a list of string values and then converts each string to its boolean equivalent using the `TO_BOOLEAN` function. For example, "true" and "TRuE" will be converted to `true`, while "false", "", "yes", and "1" will be converted to `false` or `true` according to the conversion rules.

```esql
ROW str = ["true", "TRuE", "false", "", "yes", "1"]
| EVAL bool = TO_BOOLEAN(str)
```
Convert a list of string values to their boolean equivalents.
