# NOW

Returns the current date and time.

## Syntax

`NOW()`

### Parameters

This function does not take any parameters.

## Examples

This query creates a single row containing the current date and time in a column named `current_date`.

```esql
ROW current_date = NOW()
```

This query retrieves records from the `sample_data` index where the `@timestamp` field is within the last hour, using the current time as a reference.

```esql
FROM sample_data
| WHERE @timestamp > NOW() - 1 hour
```
