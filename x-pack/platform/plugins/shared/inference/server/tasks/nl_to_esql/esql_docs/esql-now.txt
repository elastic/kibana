# NOW

Returns the current date and time.

## Syntax

`NOW()`

### Parameters

This function does not take any parameters.

## Examples

Get the current date and time:

Returns a single row containing the current date and time in a column named `current_date`.
```esql
ROW current_date = NOW()
```

Retrieve logs from the last hour:

Fetches records from the `sample_data` index where the `@timestamp` is within the last hour.
```esql
FROM sample_data
| WHERE @timestamp > NOW() - 1 hour
```
