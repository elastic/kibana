# SPACE

Returns a string consisting of a specified number of space characters.

## Syntax

`SPACE(number)`

### Parameters

#### number

The number of spaces to include in the resulting string.

## Examples

Create a string with a single space between "Hello" and "World!":

This query creates a new row with a field called `message` that contains the text "Hello World!" where "Hello" and "World!" are separated by a single space character.

```esql
ROW message = CONCAT("Hello", SPACE(1), "World!");
```
