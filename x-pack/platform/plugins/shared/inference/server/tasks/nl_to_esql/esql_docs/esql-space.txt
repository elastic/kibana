# SPACE

The SPACE function returns a string consisting of a specified number of space characters.

## Syntax

`SPACE(number)`

### Parameters

#### number

Specifies the number of spaces to include in the result string.

## Examples

Generate a string with one space between "Hello" and "World!":

This query creates a single row with a column named `message` that contains the text "Hello World!", inserting exactly one space character between "Hello" and "World!" by using the SPACE function.

```esql
ROW message = CONCAT("Hello", SPACE(1), "World!");
```
