# IP_PREFIX

The IP_PREFIX function truncates an IP address to a specified prefix length.

## Syntax

`IP_PREFIX(ip, prefixLengthV4, prefixLengthV6)`

### Parameters

#### ip

IP address of type `ip`. Both IPv4 and IPv6 addresses are supported.

#### prefixLengthV4

Prefix length for IPv4 addresses.

#### prefixLengthV6

Prefix length for IPv6 addresses.

## Examples

This query creates two rows: one with an IPv4 address (`1.2.3.4`) and one with an IPv6 address (`fe80::cae2:65ff:fece:feb9`). It then truncates the IPv4 address to a /24 prefix and the IPv6 address to a /112 prefix, producing the corresponding network prefixes for each IP.

```esql
ROW ip4 = to_ip("1.2.3.4"), ip6 = TO_IP("fe80::cae2:65ff:fece:feb9")
| EVAL ip4_prefix = IP_PREFIX(ip4, 24, 0), ip6_prefix = IP_PREFIX(ip6, 0, 112);
```
