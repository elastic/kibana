# URL_DECODE

Decodes a URL-encoded string. If the input cannot be decoded, returns `null` and adds a warning header to the response.

## Syntax

`URL_DECODE(string)`

### Parameters

#### string

The URL-encoded string to decode.

## Examples

Decode a URL-encoded string:

This query creates a row with a column `u` containing a URL-encoded string, then decodes the value in `u` using the `URL_DECODE` function.

```esql
ROW u = "https%3A%2F%2Fexample.com%2F%3Fx%3Dfoo%20bar%26y%3Dbaz"
| EVAL u = URL_DECODE(u)
```
This example decodes the URL-encoded value in the column `u`.
