# TO_STRING

Converts an input value into a string. The input can be a single value, multiple values, or an expression.

## Syntax

`TO_STRING(field)`

### Parameters

#### field

The value to convert to a string. This can be a single- or multi-valued column, or an expression.

## Examples

Converts the integer value in column `a` to a string and stores it in column `j`.

```esql
ROW a=10
| EVAL j = TO_STRING(a)
```

Converts the array of integers in column `a` to a string representation and stores it in column `j`.

```esql
ROW a=[10, 9, 8]
| EVAL j = TO_STRING(a)
```
