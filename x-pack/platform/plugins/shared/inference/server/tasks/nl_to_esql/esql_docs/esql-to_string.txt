# TO_STRING

Converts an input value into a string.

## Syntax

`TO_STRING(field)`

### Parameters

#### `field`

The input value to be converted. It can be a single- or multi-valued column or an expression.

## Examples

### Example 1: Converting a single value to a string
```esql
ROW a=10
| EVAL j = TO_STRING(a)
```

### Example 2: Converting a multivalued field to a string
```esql
ROW a=[10, 9, 8]
| EVAL j = TO_STRING(a)
```
