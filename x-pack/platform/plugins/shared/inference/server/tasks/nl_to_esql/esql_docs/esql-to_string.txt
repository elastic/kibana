# TO_STRING

Converts an input value into a string. The input can be a single value, multiple values, or an expression.

## Syntax

`TO_STRING(field)`

### Parameters

#### field

The value to convert to a string. This can be a single- or multi-valued column, or an expression.

## Examples

Convert a single integer value to a string:

This query creates a row with a single integer value `a` set to 10, then uses the `TO_STRING` function to convert the integer value to a string and assigns it to the new column `j`.

```esql
ROW a=10
| EVAL j = TO_STRING(a)
```

Convert a list of integer values to a string:

This query creates a row with a list of integers `[10, 9, 8]` in column `a`, then converts the entire list to a string representation and stores it in the column `j`.

```esql
ROW a=[10, 9, 8]
| EVAL j = TO_STRING(a)
```
