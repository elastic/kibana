# TAN

The TAN function returns the tangent of a given angle.

## Syntax

`TAN(angle)`

### Parameters

#### angle

An angle in radians. If the value is `null`, the function returns `null`.

## Examples

The following query creates a row with a column `a` set to 1.8, then calculates the tangent of `a` and stores the result in a new column called `tan`:

```esql
ROW a=1.8
| EVAL tan=TAN(a)
```
