# TAN

The `TAN` function calculates the tangent of a given angle, expressed in radians.

## Syntax

`TAN(angle)`

### Parameters

#### `angle`

An angle, in radians. If `null`, the function returns `null`.

## Examples

```esql
ROW a=1.8
| EVAL tan = TAN(a)
```

Calculate the tangent of the angle `1.8` radians and store the result in a new column named `tan`.
