# MV_DEDUPE

Remove duplicate values from a multivalued field.
<note>
  `MV_DEDUPE` may, but wonâ€™t always, sort the values in the column.
</note>

**Syntax**
`MV_DEDUPE(field)`

### Parameters

#### `field`

Multivalue expression.

**Supported types**

**Example**
```esql
ROW a=["foo", "foo", "bar", "foo"]
| EVAL dedupe_a = MV_DEDUPE(a)
```
