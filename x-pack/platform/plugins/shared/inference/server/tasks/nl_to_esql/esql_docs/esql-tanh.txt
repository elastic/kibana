# TANH

The TANH function returns the hyperbolic tangent of a given number.

## Syntax

`TANH(number)`

### Parameters

#### number

Numeric expression for which to calculate the hyperbolic tangent. If the value is `null`, the function returns `null`.

## Examples

The following query creates a single row with a column `a` set to 1.8, then calculates the hyperbolic tangent of `a` and stores the result in a new column called `tanh`:

```esql
ROW a=1.8
| EVAL tanh=TANH(a)
```
