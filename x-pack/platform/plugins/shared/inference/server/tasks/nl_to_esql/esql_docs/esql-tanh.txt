# TANH

Returns the hyperbolic tangent of a number.

## Syntax

`TANH(number)`

### Parameters

#### number

Numeric expression. If `null`, the function returns `null`.

## Examples

Calculates the hyperbolic tangent of the value in column `a` and stores the result in a new column called `tanh`:

```esql
ROW a=1.8
| EVAL tanh=TANH(a)
```
