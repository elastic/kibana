# MV_APPEND

Concatenates the values of two multi-value fields into a single field.

## Syntax

`MV_APPEND(field1, field2)`

### Parameters

#### `field1`

The first multi-value field to concatenate.

#### `field2`

The second multi-value field to concatenate.
