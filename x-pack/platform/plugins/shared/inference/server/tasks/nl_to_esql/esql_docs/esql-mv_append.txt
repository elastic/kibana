# MV_APPEND

Concatenates values of two multi-value fields.

**Syntax**
`MV_APPEND(field1, field2)`

### Parameters

#### `field1`

#### `field2`

**Supported types**

**Example**
```esql
FROM employees
| WHERE emp_no == 10039 OR emp_no == 10040
| SORT emp_no
| EVAL dates = MV_APPEND(birth_date, hire_date)
| KEEP emp_no, birth_date, hire_date, dates
```
