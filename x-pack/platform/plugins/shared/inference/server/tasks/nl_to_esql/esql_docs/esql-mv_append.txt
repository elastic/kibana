# MV_APPEND

The MV_APPEND function concatenates the values of two multi-value fields into a single multi-value field.

## Syntax

`MV_APPEND(field1, field2)`

### Parameters

#### field1

The first multi-value field whose values will be concatenated.

#### field2

The second multi-value field whose values will be concatenated.

## Examples

Concatenates the birth_date and hire_date fields into a new multi-value field called dates for employees with emp_no 10039 or 10040, and displays the relevant fields.

```esql
FROM employees
| WHERE emp_no == 10039 OR emp_no == 10040
| SORT emp_no
| EVAL dates = MV_APPEND(birth_date, hire_date)
| KEEP emp_no, birth_date, hire_date, dates
```
