# MV_APPEND

The MV_APPEND function concatenates the values of two multi-value fields.

## Syntax

`MV_APPEND(field1, field2)`

### Parameters

#### field1

The first multi-value field whose values will be concatenated.

#### field2

The second multi-value field whose values will be concatenated.

## Examples

Concatenate the `birth_date` and `hire_date` fields for selected employees:

This query selects employees with `emp_no` 10039 or 10040, sorts them by employee number, and creates a new field called `dates` that combines the values from the `birth_date` and `hire_date` fields for each employee. The output only includes the employee number, birth date, hire date, and the new `dates` field.

```esql
FROM employees
| WHERE emp_no == 10039 OR emp_no == 10040
| SORT emp_no
| EVAL dates = MV_APPEND(birth_date, hire_date)
| KEEP emp_no, birth_date, hire_date, dates
```

This example creates a new field `dates` by appending the values from `birth_date` and `hire_date` for each employee.
