# ACOS

The ACOS function returns the arccosine of a number as an angle in radians.

## Syntax

`ACOS(number)`

### Parameters

#### number

A number between -1 and 1. If the value is `null`, the function returns `null`.

## Examples

This query creates a single row with a column `a` set to 0.9, then calculates the arccosine of `a` and stores the result in a new column called `acos`.

```esql
ROW a=.9
| EVAL acos=ACOS(a)
```
