# CONTAINS

Returns a boolean value indicating whether a keyword substring exists within another string. If either parameter is null, the function returns `null`.

## Syntax

`CONTAINS(string, substring)`

### Parameters

#### string

The input string to check against. If `null`, the function returns `null`.

#### substring

The substring to search for within the input string. If `null`, the function returns `null`.

## Examples

This query creates a row with the value "hello" and then checks if the substring "ll" exists within that string. The result is stored in the column `has_ll` as a boolean value.

```esql
ROW a = "hello"
| EVAL has_ll = CONTAINS(a, "ll")
```

Checks if the substring "ll" is present in the string "hello".
