# REPEAT

Returns a string created by repeating the given string a specified number of times.

## Syntax

`REPEAT(string, number)`

### Parameters

#### string

The string expression to be repeated.

#### number

The number of times to repeat the string.

## Examples

Generates a row with the value "Hello!" and creates a new column containing "Hello!" repeated three times.

```esql
ROW a = "Hello!"
| EVAL triple_a = REPEAT(a, 3)
```
