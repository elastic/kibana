# REPEAT

The REPEAT function returns a string created by concatenating a given string with itself a specified number of times.

## Syntax

`REPEAT(string, number)`

### Parameters

#### string

The string expression to be repeated.

#### number

The number of times to repeat the string.

## Examples

This query creates a row with the value "Hello!" in column `a`, then uses the REPEAT function to concatenate the value of `a` three times, resulting in "Hello!Hello!Hello!" in the new column `triple_a`.

```esql
ROW a = "Hello!"
| EVAL triple_a = REPEAT(a, 3)
```
