# REPEAT

Returns a string constructed by concatenating `string` with itself the specified `number` of times.

**Syntax**
`REPEAT(string, number)`

### Parameters

#### `string`

String expression.

#### `number`

Number times to repeat.

**Supported types**

**Example**
```esql
ROW a = "Hello!"
| EVAL triple_a = REPEAT(a, 3)
```
