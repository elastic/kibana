# REPEAT

Returns a string created by concatenating the input string with itself a specified number of times.

## Syntax

`REPEAT(string, number)`

### Parameters

#### string

The string expression to be repeated.

#### number

The number of times to repeat the string.

## Examples

The following query creates a row with the value "Hello!" in column `a`, then uses the `REPEAT` function to concatenate "Hello!" three times and stores the result in a new column called `triple_a`:

```esql
ROW a = "Hello!"
| EVAL triple_a = REPEAT(a, 3)
```
