# RIGHT

Return the substring that extracts *length* chars from *str* starting from the right.

**Syntax**
`RIGHT(string, length)`

### Parameters

#### `string`

The string from which to returns a substring.

#### `length`

The number of characters to return.

**Supported types**

**Example**

Extracts the last three characters from the `last_name` column, sorts the results alphabetically, and limits the output to the first five rows.

```esql
FROM employees
| KEEP last_name
| EVAL right = RIGHT(last_name, 3)
```
