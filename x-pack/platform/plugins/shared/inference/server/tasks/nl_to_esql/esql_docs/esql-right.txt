# RIGHT

Returns a substring containing the specified number of characters from the right end of a string.

## Syntax

`RIGHT(string, length)`

### Parameters

#### string

The string from which to return a substring.

#### length

The number of characters to return from the right end of the string.

## Examples

Extracts the last three characters from each employee's last name and stores them in a new column called `right`.
```esql
FROM employees
| KEEP last_name
| EVAL right = RIGHT(last_name, 3)
```
