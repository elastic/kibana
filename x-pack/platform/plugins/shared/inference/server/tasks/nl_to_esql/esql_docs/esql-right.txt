# RIGHT

Returns a substring that extracts a specified number of characters from the right side of a string.

## Syntax

`RIGHT(string, length)`

### Parameters

#### `string`

The string from which to return a substring.

#### `length`

The number of characters to return.

## Examples

```esql
FROM employees
| KEEP last_name
| EVAL right = RIGHT(last_name, 3)
| SORT last_name ASC
| LIMIT 5
```

Extracts the last three characters from the `last_name` column, sorts the results alphabetically by `last_name`, and limits the output to the first five rows.
