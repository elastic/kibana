# RIGHT

Returns a substring that extracts a specified number of characters from the right side of a string.

## Syntax

`RIGHT(string, length)`

### Parameters

#### string

The string from which the substring will be extracted.

#### length

The number of characters to return from the right end of the string.

## Examples

This query selects the `last_name` column from the `employees` data, then creates a new column called `right` that contains the last three characters of each employee's last name.

```esql
FROM employees
| KEEP last_name
| EVAL right = RIGHT(last_name, 3)
```
