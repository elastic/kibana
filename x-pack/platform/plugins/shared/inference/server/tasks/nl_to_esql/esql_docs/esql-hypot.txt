# HYPOT

The HYPOT function calculates the hypotenuse of two numeric values and always returns a double. If either input is infinity, the result is null.

## Syntax

`HYPOT(number1, number2)`

### Parameters

#### number1

Numeric expression. If this value is null, the function returns null.

#### number2

Numeric expression. If this value is null, the function returns null.

## Examples

Calculates the hypotenuse of the sides `a = 3.0` and `b = 4.0`, storing the result in column `c`.

```esql
ROW a = 3.0, b = 4.0
| EVAL c = HYPOT(a, b)
```
