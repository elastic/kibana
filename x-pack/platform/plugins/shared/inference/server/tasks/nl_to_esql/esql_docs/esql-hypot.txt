# HYPOT

Returns the hypotenuse of two numbers. The input values can be any numeric type, and the result is always a double. If either input is infinity, the result is null.

## Syntax

`HYPOT(number1, number2)`

### Parameters

#### number1

Numeric expression. If this value is null, the function returns null.

#### number2

Numeric expression. If this value is null, the function returns null.

## Examples

This query creates a row with values `a = 3.0` and `b = 4.0`, then calculates the hypotenuse using these values and stores the result in column `c`.

```esql
ROW a = 3.0, b = 4.0
| EVAL c = HYPOT(a, b)
```
Calculate the hypotenuse for the sides 3.0 and 4.0.
