# HYPOT

The HYPOT function calculates the hypotenuse of two numeric values. The result is always a double. If either input is infinity, the function returns null.

## Syntax

`HYPOT(number1, number2)`

### Parameters

#### number1

Numeric expression. If this value is null, the function returns null.

#### number2

Numeric expression. If this value is null, the function returns null.

## Examples

This query creates a row with two columns, `a` and `b`, set to 3.0 and 4.0 respectively. It then calculates the hypotenuse using the HYPOT function and stores the result in a new column `c`.

```esql
ROW a = 3.0, b = 4.0
| EVAL c = HYPOT(a, b)
```
