# URL_ENCODE_COMPONENT

Encodes a string for safe use in a URL by percent-encoding all characters except alphanumerics, `.`, `-`, `_`, and `~`. Spaces are encoded as `%20`.

## Syntax

`URL_ENCODE_COMPONENT(string)`

### Parameters

#### string

The URL or string to encode.

## Examples

Encodes the value of the `u` field, which contains a URL with spaces and special characters, into a percent-encoded format suitable for use in a URL.

```esql
ROW u = "https://example.com/?x=foo bar&y=baz"
| EVAL u = URL_ENCODE_COMPONENT(u)
```
This example encodes the URL, converting spaces and special characters to their percent-encoded representations.
