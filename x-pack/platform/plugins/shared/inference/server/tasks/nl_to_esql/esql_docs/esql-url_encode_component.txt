# URL_ENCODE_COMPONENT

The URL_ENCODE_COMPONENT function encodes a string for safe inclusion in a URL component. All characters are percent-encoded except for alphanumerics, `.`, `-`, `_`, and `~`. Spaces are encoded as `%20`.

## Syntax

`URL_ENCODE_COMPONENT(string)`

### Parameters

#### string

The string value to encode for use in a URL component.

## Examples

Encode a URL string, converting spaces and special characters to their percent-encoded form:

This query creates a row with a URL string, then encodes the string so that spaces and special characters are replaced with their percent-encoded equivalents, making it safe for use in a URL component.

```esql
ROW u = "https://example.com/?x=foo bar&y=baz"
| EVAL u = URL_ENCODE_COMPONENT(u)
```
