# TO_GEOTILE

The TO_GEOTILE function converts an input value to a `geotile` value. Conversion is successful only if the input string matches the `geotile` format, as defined for the geotile grid aggregation.

## Syntax

`TO_GEOTILE(field)`

### Parameters

#### field

The input value to be converted. This can be a single- or multi-valued column, or an expression.

## Examples

Converts the string "4/8/5" into a geotile value and adds it as a new column.

```esql
ROW string = "4/8/5"
| EVAL geotile = TO_GEOTILE(string)
```
