# REVERSE

Returns a new string with the characters of the input string in reverse order.

## Syntax

`REVERSE(str)`

### Parameters

#### str

The string expression to reverse. If the value is `null`, the function returns `null`.

## Examples

Reverses the value of the `message` field from "Some Text" to "txeT emoS".

```esql
ROW message = "Some Text" | EVAL message_reversed = REVERSE(message);
```
