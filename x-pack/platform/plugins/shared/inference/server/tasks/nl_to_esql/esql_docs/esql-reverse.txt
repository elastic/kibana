# REVERSE

The `REVERSE` function returns a new string with the characters of the input string in reverse order.

## Syntax

`REVERSE(str)`

### Parameters

#### `str`

String expression. If `null`, the function returns `null`.

## Examples

Reversing a simple string

```esql
ROW message = "Some Text"
| EVAL message_reversed = REVERSE(message);
```

| message   | message_reversed |
|-----------|------------------|
| Some Text | txeT emoS        |

Reversing a string with emojis

ROW bending_arts = "ğŸ’§ğŸª¨ğŸ”¥ğŸ’¨"

`REVERSE` works with Unicode and preserves grapheme clusters during reversal.

## Limitations

If Elasticsearch is running with a JDK version less than 20, the function may not properly reverse grapheme clusters. For example, "ğŸ‘ğŸ½ğŸ˜Š" might be reversed to "ğŸ½ğŸ‘ğŸ˜Š" instead of the correct "ğŸ˜ŠğŸ‘ğŸ½". Elastic Cloud and the JDK bundled with Elasticsearch use newer JDKs, so this issue typically arises only if an older JDK is explicitly used.
