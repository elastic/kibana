# REVERSE

Returns a new string with the characters of the input string in reverse order. Unicode grapheme clusters are preserved during reversal.

## Syntax

`REVERSE(str)`

### Parameters

#### str

String expression to be reversed. If the value is `null`, the function returns `null`.

## Examples

Reverse the characters in a string:

This query creates a row with a column named `message` containing the text "Some Text", then adds a new column `message_reversed` with the characters of `message` in reverse order.

```esql
ROW message = "Some Text" | EVAL message_reversed = REVERSE(message);
```

Reverse a string containing unicode grapheme clusters, keeping them together:

This query creates a row with a column named `bending_arts` containing the string "ğŸ’§ğŸª¨ğŸ”¥ğŸ’¨", then adds a new column `bending_arts_reversed` where the order of the grapheme clusters (emojis) is reversed, but each emoji remains intact.

```esql
ROW bending_arts = "ğŸ’§ğŸª¨ğŸ”¥ğŸ’¨" | EVAL bending_arts_reversed = REVERSE(bending_arts);
```
