# REVERSE

Returns a new string representing the input string in reverse order.

**Syntax**
`REVERSE(str)`

### Parameters

#### `str`

String expression. If `null`, the function returns `null`.

**Supported types**

**Examples**
```esql
ROW message = "Some Text" | EVAL message_reversed = REVERSE(message);
```
