# LENGTH

Returns the character length of a string.
<note>
  All strings are in UTF-8, so a single character can use multiple bytes.
</note>

**Syntax**
`LENGTH(string)`

### Parameters

#### `string`

String expression. If `null`, the function returns `null`.

**Supported types**

**Example**
```esql
FROM airports
| WHERE country == "India"
| KEEP city
| EVAL fn_length = LENGTH(city)
```
