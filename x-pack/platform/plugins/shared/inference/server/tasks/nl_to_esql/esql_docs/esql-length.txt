# LENGTH

Returns the character length of a string. All strings are in UTF-8, so a single character can use multiple bytes.

## Syntax

`LENGTH(string)`

### Parameters

#### string

String expression. If the value is `null`, the function returns `null`.

## Examples

Calculates the character length of each city name for airports located in India and displays the result in a new column.
```esql
FROM airports
| WHERE country == "India"
| KEEP city
| EVAL fn_length = LENGTH(city)
```
