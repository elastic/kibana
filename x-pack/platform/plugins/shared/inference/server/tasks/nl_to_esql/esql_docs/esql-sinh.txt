# SINH

Returns the hyperbolic sine of a number.

## Syntax

`SINH(number)`

### Parameters

#### number

Numeric expression. If `null`, the function returns `null`.

## Examples

This query creates a single row with a column `a` set to 1.8, then calculates the hyperbolic sine of `a` and stores the result in a new column called `sinh`.

```esql
ROW a=1.8
| EVAL sinh=SINH(a)
```
Calculates the hyperbolic sine of the value 1.8 and stores the result in the column `sinh`.
