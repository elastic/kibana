# SINH

Returns the hyperbolic sine of a number.

## Syntax

`SINH(number)`

### Parameters

#### number

Numeric expression. If `null`, the function returns `null`.

## Examples

Calculates the hyperbolic sine of the value in column `a` and stores the result in a new column `sinh`:

```esql
ROW a=1.8
| EVAL sinh=SINH(a)
```
