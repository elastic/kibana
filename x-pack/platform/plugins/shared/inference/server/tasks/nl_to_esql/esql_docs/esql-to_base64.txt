# TO_BASE64

The TO_BASE64 command encodes a string into its base64 representation.

## Syntax

`TO_BASE64(string)`

### Parameters

#### string

The string value to be encoded.

## Examples

Encodes the value "elastic" in the column `a` to its base64 representation and stores the result in column `e`:

```esql
ROW a = "elastic"
| EVAL e = TO_BASE64(a)
```
