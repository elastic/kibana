# TO_BASE64

The TO_BASE64 command encodes a string into its base64 representation.

## Syntax

`TO_BASE64(string)`

### Parameters

#### string

The string value to be encoded.

## Examples

The following query creates a row with the value "elastic" and then encodes it into its base64 representation using the TO_BASE64 function:

```esql
ROW a = "elastic"
| EVAL e = TO_BASE64(a)
```
