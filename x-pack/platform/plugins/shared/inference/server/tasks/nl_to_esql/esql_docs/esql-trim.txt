# TRIM

Removes leading and trailing whitespaces from a string.

## Syntax

`TRIM(string)`

### Parameters

#### string

String expression to be trimmed. If the value is `null`, the function returns `null`.

## Examples

The following query creates two columns, `message` and `color`, each containing extra spaces at the beginning and end. It then removes the leading and trailing spaces from both columns using the `TRIM` function.

```esql
ROW message = "   some text  ",  color = " red "
| EVAL message = TRIM(message)
| EVAL color = TRIM(color)
```
