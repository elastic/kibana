# TRIM

Removes leading and trailing whitespaces from a string.

## Syntax

`TRIM(string)`

### Parameters

#### string

String expression to be trimmed. If the value is `null`, the function returns `null`.

## Examples

Removes leading and trailing spaces from the `message` and `color` columns in a single row.

```esql
ROW message = "   some text  ",  color = " red "
| EVAL message = TRIM(message)
| EVAL color = TRIM(color)
```
