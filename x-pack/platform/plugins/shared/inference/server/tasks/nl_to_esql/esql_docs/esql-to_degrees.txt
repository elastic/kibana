# TO_DEGREES

Converts a number in radians to degrees.

**Syntax**
`TO_DEGREES(number)`

### Parameters

#### `number`

Input value. The input can be a single- or multi-valued column or an expression.

**Supported types**

**Example**
```esql
ROW rad = [1.57, 3.14, 4.71]
| EVAL deg = TO_DEGREES(rad)
```
