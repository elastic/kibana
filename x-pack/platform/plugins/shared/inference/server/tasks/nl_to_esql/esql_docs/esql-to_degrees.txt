# TO_DEGREES

Converts a numeric value from radians to degrees.

## Syntax

`TO_DEGREES(number)`

### Parameters

#### number

The input value to convert from radians to degrees. This can be a single value, a multi-valued column, or an expression.

## Examples

Converts each value in the `rad` array from radians to degrees and stores the result in a new column `deg`:

```esql
ROW rad = [1.57, 3.14, 4.71]
| EVAL deg = TO_DEGREES(rad)
```
