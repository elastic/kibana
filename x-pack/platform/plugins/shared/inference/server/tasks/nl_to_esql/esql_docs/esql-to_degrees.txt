# TO_DEGREES

Converts a number in radians to degrees.

## Syntax

`TO_DEGREES(number)`

### Parameters

#### number

Input value in radians. The input can be a single- or multi-valued column or an expression.

## Examples

Convert an array of radian values to degrees:

This query creates a row with an array of radian values and then adds a new column that contains the corresponding degree values for each radian value in the array.

```esql
ROW rad = [1.57, 3.14, 4.71]
| EVAL deg = TO_DEGREES(rad)
```
