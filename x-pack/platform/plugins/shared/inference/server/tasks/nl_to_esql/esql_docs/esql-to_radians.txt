# TO_RADIANS

Converts a number from degrees to radians.

## Syntax

`TO_RADIANS(number)`

### Parameters

#### number

Input value to be converted from degrees to radians. The input can be a single- or multi-valued column or an expression.

## Examples

Convert an array of degree values to radians:

This query creates a row with a column `deg` containing an array of degree values `[90.0, 180.0, 270.0]`, then adds a new column `rad` that contains the corresponding values converted to radians.

```esql
ROW deg = [90.0, 180.0, 270.0]
| EVAL rad = TO_RADIANS(deg)
```
