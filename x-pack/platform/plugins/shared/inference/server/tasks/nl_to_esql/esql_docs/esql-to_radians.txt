# TO_RADIANS

Converts a number in degrees to radians.

**Syntax**
`TO_RADIANS(number)`

### Parameters

#### `number`

Input value. The input can be a single- or multi-valued column or an expression.

**Supported types**

**Example**
```esql
ROW deg = [90.0, 180.0, 270.0]
| EVAL rad = TO_RADIANS(deg)
```
