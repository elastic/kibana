# TO_GEOHEX

The TO_GEOHEX function converts an input value to a `geohex` value. Conversion is successful only if the input string matches the `geohex` format, as defined for the geohex grid aggregation.

## Syntax

`TO_GEOHEX(field)`

### Parameters

#### field

The input value to convert. This can be a single- or multi-valued column, or an expression.

## Examples

Converts the string `"841f059ffffffff"` into a geohex value.

```esql
ROW string = "841f059ffffffff"
| EVAL geohex = TO_GEOHEX(string)
```
