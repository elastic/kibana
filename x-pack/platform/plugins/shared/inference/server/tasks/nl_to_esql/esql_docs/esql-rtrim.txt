# RTRIM

Removes trailing whitespaces from a string.

## Syntax

`RTRIM(string)`

### Parameters

#### string

String expression to process. If `null`, the function returns `null`.

## Examples

This query creates two columns, `message` and `color`, with leading and trailing spaces. It then removes trailing spaces from both columns using the `RTRIM` function, and finally wraps the cleaned values in single quotes for clarity.

```esql
ROW message = "   some text  ",  color = " red "
| EVAL message = RTRIM(message)
| EVAL color = RTRIM(color)
| EVAL message = CONCAT("'", message, "'")
| EVAL color = CONCAT("'", color, "'")
```

Remove trailing spaces from the `message` and `color` columns, then wrap the results in single quotes.
