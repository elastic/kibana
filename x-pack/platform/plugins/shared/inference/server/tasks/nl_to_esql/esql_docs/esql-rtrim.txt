# RTRIM

Removes trailing whitespaces from a string.

## Syntax

`RTRIM(string)`

### Parameters

#### string

The string expression to process. If the value is `null`, the function returns `null`.

## Examples

The following query demonstrates how to remove trailing spaces from the `message` and `color` columns, and then wrap the cleaned results in single quotes:

```esql
ROW message = "   some text  ",  color = " red "
| EVAL message = RTRIM(message)
| EVAL color = RTRIM(color)
| EVAL message = CONCAT("'", message, "'")
| EVAL color = CONCAT("'", color, "'")
```

This query creates a row with two columns: `message` containing `"   some text  "` and `color` containing `" red "`. It then removes any trailing spaces from both columns using the `RTRIM` function. Finally, it wraps the trimmed values in single quotes using the `CONCAT` function.
