# CEIL

Rounds a number up to the nearest integer. For `double` values, it returns the closest `double` value to the integer, similar to `Math.ceil`. For `long` (including unsigned) and `integer` types, this function does nothing.

## Syntax

`CEIL(number)`

### Parameters

#### number

Numeric expression to be rounded up. If the value is `null`, the function returns `null`.

## Examples

The following query creates a row with a column `a` set to 1.8, then uses the `CEIL` function to round the value up to the nearest integer, resulting in 2.

```esql
ROW a=1.8
| EVAL a=CEIL(a)
```
