# CEIL

Rounds a number up to the nearest integer. For `double` values, it selects the closest `double` value to the integer, similar to `Math.ceil`. For `long` (including unsigned) and `integer`, this function is a no-op.

## Syntax

`CEIL(number)`

### Parameters

#### `number`

Numeric expression. If `null`, the function returns `null`.

## Examples

### Example 1: Rounding up a decimal value

```esql
ROW a=1.8
| EVAL a = CEIL(a)
```

This example rounds the value `1.8` up to the nearest integer, resulting in `2`.
