## COS

The `COS` function calculates the cosine of a given angle in radians.

## Syntax

`COS(angle)`

### Parameters

#### `angle`

An angle, in radians. If `null`, the function returns `null`.

## Examples

```esql
ROW a=1.8
| EVAL cos = COS(a)
```

Calculate the cosine of the angle `1.8` radians and store the result in a new column named `cos`.
