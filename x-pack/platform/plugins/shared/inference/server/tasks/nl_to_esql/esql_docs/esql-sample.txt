# SAMPLE

The SAMPLE function collects a specified number of sample values from a field.

## Syntax

`SAMPLE(field, limit)`

### Parameters

#### field

The field to collect sample values for.

#### limit

The maximum number of values to collect.

## Examples

This query retrieves up to 5 sample values from the `gender` field in the `employees` dataset.

```esql
FROM employees
| STATS sample = SAMPLE(gender, 5)
```
Collect up to 5 sample values from the `gender` field in the `employees` dataset.
```
