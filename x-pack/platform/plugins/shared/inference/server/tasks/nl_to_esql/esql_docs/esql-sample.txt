# SAMPLE

The SAMPLE function collects a specified number of sample values from a field.

## Syntax

`SAMPLE(field, limit)`

### Parameters

#### field

The field from which to collect sample values.

#### limit

The maximum number of values to collect.

## Examples

The following query retrieves up to 5 sample values from the `gender` field in the `employees` data:

```esql
FROM employees
| STATS sample = SAMPLE(gender, 5)
```
| SAMPLE 0.05
