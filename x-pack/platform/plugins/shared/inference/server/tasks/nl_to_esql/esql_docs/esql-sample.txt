# SAMPLE

Collects sample values for a field.

```
stack: ga 9.1.0
```

**Syntax**
`SAMPLE(field, limit)`

### Parameters

#### `field`

The field to collect sample values for.

#### `limit`

The maximum number of values to collect.

**Supported types**

**Example**
```esql
FROM employees
| STATS sample = SAMPLE(gender, 5)
```
