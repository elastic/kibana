# MV_MEDIAN

Converts a multivalued field into a single-valued field containing the median value.

## Syntax

`MV_MEDIAN(number)`

### Parameters

#### `number`

A multivalue expression.

## Examples

### Example 1

Calculate the median of a multivalued column:

```esql
ROW a=[3, 5, 1]
| EVAL median_a = MV_MEDIAN(a)
```

### Example 2

When the column has an even number of values, the result is the average of the middle two entries. If the column is not a floating-point type, the average rounds **down**:

```esql
ROW a=[3, 7, 1, 6]
| EVAL median_a = MV_MEDIAN(a)
```
