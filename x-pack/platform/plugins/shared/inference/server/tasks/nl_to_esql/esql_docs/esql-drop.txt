## DROP

The `DROP` processing command removes one or more columns from the result set.

## Syntax

`DROP columns`

### Parameters

#### `columns`

A comma-separated list of columns to remove. Supports wildcards.

## Examples

#### Example 1: Removing a specific column
Remove the `height` column from the result set:
```esql
FROM employees
| DROP height
```

#### Example 2: Removing columns using wildcards
Use wildcards to drop all columns with names that match a specific pattern, such as all columns starting with `height`:
```esql
FROM employees
| DROP height*
```
