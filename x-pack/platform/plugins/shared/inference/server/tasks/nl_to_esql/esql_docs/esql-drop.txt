# DROP

The DROP command removes one or more columns from the result set.

## Syntax

`DROP columns`

### Parameters

#### columns

A comma-separated list of columns to remove. Wildcards are supported to match multiple columns.

## Examples

Remove the column named `height`:

This query retrieves data from the `employees` index and removes the `height` column from the results.

```esql
FROM employees
| DROP height
```

Remove all columns with names that start with `height` using a wildcard:

This query retrieves data from the `employees` index and removes all columns whose names begin with `height`, such as `height_cm` or `height_in`, from the results.

```esql
FROM employees
| DROP height*
```
