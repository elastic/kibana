# DROP

The `DROP` command removes one or more columns from the result set.

## Syntax

`DROP columns`

### Parameters

#### columns

A comma-separated list of columns to remove. Supports wildcards.

## Examples

### Example 1

Remove a specific column:

```esql
FROM employees
| DROP height
```

### Example 2

Use wildcards to drop all columns with names matching a pattern:

```esql
FROM employees
| DROP height*
```
