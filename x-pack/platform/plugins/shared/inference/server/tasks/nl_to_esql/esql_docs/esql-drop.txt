# DROP

The DROP command removes one or more columns from the result set.

## Syntax

`DROP columns`

### Parameters

#### columns

A comma-separated list of columns to remove. Wildcards are supported to match multiple columns.

## Examples

Remove the column named `height`:

This query retrieves all records from the `employees` source and removes the `height` column from the results.

```esql
FROM employees
| DROP height
```

Remove all columns with names that start with `height` using a wildcard:

This query retrieves all records from the `employees` source and removes any columns whose names begin with `height` (such as `height`, `height_cm`, `height_in`, etc.) from the results.

```esql
FROM employees
| DROP height*
```
