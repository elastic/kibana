# DROP

The DROP command removes one or more columns from the result set. You can specify columns by name or use wildcards to match multiple columns.

## Syntax

`DROP columns`

### Parameters

#### columns

A comma-separated list of columns to remove. Wildcards are supported to match multiple columns.

## Examples

Removes the column named `height` from the employees dataset.

```esql
FROM employees
| DROP height
```

Removes all columns from the employees dataset whose names start with `height` by using a wildcard.

```esql
FROM employees
| DROP height*
```
