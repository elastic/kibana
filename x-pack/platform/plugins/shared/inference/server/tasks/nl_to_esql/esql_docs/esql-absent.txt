# ABSENT

The ABSENT function returns true if the input expression yields no non-null values within the current aggregation context; otherwise, it returns false.

## Syntax

`ABSENT(field)`

### Parameters

#### field

Expression that outputs values to be checked for absence.

## Examples

Checks whether the `languages` field is missing for the employee with `emp_no` 10020.

```esql
FROM employees
| WHERE emp_no == 10020
| STATS is_absent = ABSENT(languages)
```

Determines if there are any missing `salary` values within each group of `languages`.

```esql
FROM employees
| STATS is_absent = ABSENT(salary) BY languages
```

Returns 1 if the `languages` field is absent for the employee with `emp_no` 10020, and 0 otherwise.

```esql
FROM employees
| WHERE emp_no == 10020
| STATS is_absent = TO_INTEGER(ABSENT(languages))
```
