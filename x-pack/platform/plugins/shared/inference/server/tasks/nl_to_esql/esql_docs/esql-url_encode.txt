# URL_ENCODE

URL-encodes the input. All characters are percent-encoded except for alphanumerics, `.`, `-`, `_`, and `~`. Spaces are encoded as `+`.

```
stack: ga 9.2.0
```

**Syntax**
`URL_ENCODE(string)`

### Parameters

#### `string`

The URL to encode.

**Supported types**

**Example**
```esql
ROW u = "https://example.com/?x=foo bar&y=baz" | EVAL u = URL_ENCODE(u)
```
