# URL_ENCODE

The URL_ENCODE function encodes a string for safe use in URLs by percent-encoding all characters except alphanumerics, `.`, `-`, `_`, and `~`. Spaces are encoded as `+`.

## Syntax

`URL_ENCODE(string)`

### Parameters

#### string

The string value to encode as a URL.

## Examples

Encodes the given URL string, replacing spaces with `+` and special characters with percent-encoded values.

```esql
ROW u = "https://example.com/?x=foo bar&y=baz" | EVAL u = URL_ENCODE(u)
```
This example encodes the URL, converting spaces to `+` and percent-encoding special characters.
