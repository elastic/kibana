# MV_CONCAT

The MV_CONCAT function converts a multivalued string expression into a single value by concatenating all values, separated by a specified delimiter.

## Syntax

`MV_CONCAT(string, delim)`

### Parameters

#### string

A multivalue expression containing the values to concatenate.

#### delim

The delimiter used to separate the concatenated values.

## Examples

The following query creates a row with a multivalued string column `a` containing the values "foo", "zoo", and "bar". It then concatenates these values into a single string, separated by a comma and space, and stores the result in column `j`:

```esql
ROW a=["foo", "zoo", "bar"]
| EVAL j = MV_CONCAT(a, ", ")
```

This query demonstrates how to concatenate values from a non-string multivalued column. It first converts the integer values in column `a` to strings, then joins them into a single string separated by a comma and space, storing the result in column `j`:

```esql
ROW a=[10, 9, 8]
| EVAL j = MV_CONCAT(TO_STRING(a), ", ")
```
