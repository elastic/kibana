# COUNT_OVER_TIME

Calculates the count over time for a specified field within a given time window.

## Syntax

`COUNT_OVER_TIME(field, window)`

### Parameters

#### field

The metric field to calculate the count over time for.

#### window

The time window over which to compute the count over time.

## Examples

Calculates the number of occurrences of `network.cost` in each 1-minute interval, grouped by cluster.

```esql
TS k8s
| STATS count=COUNT(COUNT_OVER_TIME(network.cost))
  BY cluster, time_bucket = BUCKET(@timestamp,1minute)
```
