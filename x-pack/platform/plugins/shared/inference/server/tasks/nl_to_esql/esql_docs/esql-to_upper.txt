# TO_UPPER

Returns a new string with all characters in the input string converted to upper case.

## Syntax

`TO_UPPER(str)`

### Parameters

#### str

String expression to be converted to upper case. If the value is `null`, the function returns `null`. The input can be a single-valued or multi-valued column or expression.

## Examples

Convert the contents of the `message` column to upper case:

This query creates a row with a `message` column containing the text "Some Text", then adds a new column `message_upper` where the value of `message` is converted to all upper case letters.

```esql
ROW message = "Some Text"
| EVAL message_upper = TO_UPPER(message)
```
