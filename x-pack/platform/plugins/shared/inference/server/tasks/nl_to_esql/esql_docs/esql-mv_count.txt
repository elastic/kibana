# MV_COUNT

The MV_COUNT function converts a multivalued expression into a single-valued column that contains the count of values.

## Syntax

`MV_COUNT(field)`

### Parameters

#### field

A multivalue expression to be counted.

## Examples

Count the number of values in a multivalued column:

```esql
ROW a=["foo", "zoo", "bar"]
| EVAL count_a = MV_COUNT(a)
```
Counts how many items are present in the array `a` and saves that number in the new column `count_a`.
