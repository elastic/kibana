# MV_COUNT

Converts a multivalued expression into a single valued column containing a count of the number of values.

**Syntax**
`MV_COUNT(field)`

### Parameters

#### `field`

Multivalue expression.

**Supported types**

**Example**
```esql
ROW a=["foo", "zoo", "bar"]
| EVAL count_a = MV_COUNT(a)
```
