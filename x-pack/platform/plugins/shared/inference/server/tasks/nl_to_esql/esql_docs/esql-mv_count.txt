# MV_COUNT

The MV_COUNT function converts a multivalued expression into a single valued column that contains the count of values in each row.

## Syntax

`MV_COUNT(field)`

### Parameters

#### field

A multivalue expression to be counted.

## Examples

Count the number of values in the array column `a`:

This query creates a row with an array column `a` containing three string values: "foo", "zoo", and "bar". It then uses the `MV_COUNT` function to count how many values are in the array and stores the result in a new column called `count_a`.

```esql
ROW a=["foo", "zoo", "bar"]
| EVAL count_a = MV_COUNT(a)
```
