# TO_TIMEDURATION

Converts an input value into a `time_duration` value.

## Syntax

`TO_TIMEDURATION(field)`

### Parameters

#### `field`

The input value. Must be a valid constant time duration expression.

## Examples

### Example 1: Adding and subtracting time durations

```esql
ROW x = "2024-01-01"::datetime
| EVAL y = x + "3 hours"::time_duration, z = x - TO_TIMEDURATION(`3 hours`)
```

This example demonstrates how to add and subtract a time duration from a datetime value. The `TO_TIMEDURATION` function is used to convert the string `"3 hours"` into a `time_duration` value.
