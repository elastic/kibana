# TO_TIMEDURATION

Converts an input value into a `time_duration` value.

## Syntax

`TO_TIMEDURATION(field)`

### Parameters

#### field

Input value. The input must be a valid constant time duration expression.

## Examples

Calculate the difference between a datetime and a time duration:

Adds and subtracts a three-hour time duration to a given datetime value, demonstrating both direct and function-based duration handling.

```esql
ROW x = "2024-01-01"::datetime
| EVAL y = x + "3 hours"::time_duration, z = x - TO_TIMEDURATION("3 hours");
```
