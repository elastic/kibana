# SIN

The SIN function returns the sine of a given angle.

## Syntax

`SIN(angle)`

### Parameters

#### angle

An angle in radians. If the value is `null`, the function returns `null`.

## Examples

The following query creates a single row with a column `a` set to 1.8, then calculates the sine of that value and stores the result in a new column called `sin`:

```esql
ROW a=1.8
| EVAL sin=SIN(a)
```
