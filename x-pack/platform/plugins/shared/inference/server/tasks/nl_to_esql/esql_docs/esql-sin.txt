# SIN

Returns the sine of an angle, where the angle is specified in radians.

## Syntax

`SIN(angle)`

### Parameters

#### angle

An angle in radians. If the value is `null`, the function returns `null`.

## Examples

This query creates a single row with a column `a` set to 1.8, then calculates the sine of `a` and stores the result in a new column called `sin`:

```esql
ROW a=1.8
| EVAL sin=SIN(a)
```
