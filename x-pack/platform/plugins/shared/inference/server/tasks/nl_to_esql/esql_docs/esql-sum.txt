# SUM

The `SUM` function calculates the total of a numeric expression.

## Syntax

`SUM(number)`

### Parameters

#### `number`

A numeric expression to be summed.

## Examples

#### Example 1: Summing a field
Calculate the total number of languages spoken by employees:

```esql
FROM employees
| STATS SUM(languages)
```
