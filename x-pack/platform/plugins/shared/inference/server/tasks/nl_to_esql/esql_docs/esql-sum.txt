# SUM

The `SUM` function calculates the total of a numeric expression.

## Syntax

`SUM(number)`

### Parameters

#### `number`

A numeric expression to be summed.

## Examples

#Summing a field

```esql
FROM employees
| STATS SUM(languages)
```

Calculate the total number of languages across all employees.

#Using inline functions

```esql
FROM employees
| STATS total_salary_changes = SUM(MV_MAX(salary_change))
```

Calculate the total of each employeeâ€™s maximum salary changes by applying the `MV_MAX` function to each row and summing the results.
