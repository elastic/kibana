# Cloud Links

Adds links and UI controls to the Elastic Cloud console, including items in the user menu and help menu.

## User menu items

The plugin registers the following items in the user menu (the popover behind the user avatar in the top navigation):

- **Profile** — Link to the Cloud profile page.
- **Billing** — Link to the Cloud billing page (when the user has access).
- **Organization** — Link to the Cloud organization page.
- **Appearance** — Opens a modal for changing color mode (light/dark/system) and interface contrast (normal/high).
- **Language** — Opens a modal for selecting the display language. The selected locale is saved to the user profile and takes effect after a page reload.

## Help menu items

- **Connection details** — Opens a flyout showing the deployment's Elasticsearch endpoint, Cloud ID, and API key management.
