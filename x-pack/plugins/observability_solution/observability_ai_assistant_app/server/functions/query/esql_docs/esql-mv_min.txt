MV_MIN

Syntax
Parameters
v
Multivalue expression.
DescriptionConverts a multivalued expression into a single valued column containing the
minimum value.Supported types
Examples
```esql
ROW a=[2, 1]
| EVAL min_a = MV_MIN(a)
```

It can be used by any column type, including keyword columns. In that case,
it picks the first string, comparing their utf-8 representation byte by byte:
```esql
ROW a=["foo", "bar"]
| EVAL min_a = MV_MIN(a)
```
