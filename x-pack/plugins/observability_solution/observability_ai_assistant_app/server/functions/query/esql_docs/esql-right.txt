RIGHT

Syntax
Parameters
str
The string from which to returns a substring.
length
The number of characters to return.
DescriptionReturn the substring that extracts length chars from str starting
from the right.Supported types
Example
```esql
FROM employees
| KEEP last_name
| EVAL right = RIGHT(last_name, 3)
| SORT last_name ASC
| LIMIT 5
```
