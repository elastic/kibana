CEIL
[]
Round a number up to the nearest integer.
    ROW a=1.8
    | EVAL a=CEIL(a)
  a:double
  ----------
  2
This is a noop for long (including unsigned) and integer. For double
this picks the the closest double value to the integer ala Math.ceil.
Supported types:
  n               result
  --------------- ---------------
  double          double
  integer         integer
  long            long
  unsigned_long   unsigned_long
