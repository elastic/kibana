Custom ESLint rules enforcing UI conventions for the `@kbn/alerting-v2-rule-form` package.

## `@kbn/alerting-v2/require_eui_form_compressed`

Requires an explicit `compressed` prop on EUI form controls (`EuiFieldText`, `EuiFieldNumber`, `EuiSelect`, `EuiComboBox`, `EuiSuperSelect`, `EuiTextArea`, `EuiSuperDatePicker`) to ensure compact form layouts.
