`@kbn/eslint-plugin-packages` is an ESLint plugin providing custom rules to allow us to enforce specific behaviours we need to implement Bazel packages across the monorepo as our main development unit.

## `@kbn/packages/no_naked_eslint_disable`

Disallows the usage of naked eslint-disable comments without being specific about each rule to disable.
