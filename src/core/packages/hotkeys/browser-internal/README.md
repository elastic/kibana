# @kbn/core-hotkeys-browser-internal

This package contains the implementation and internal types for core's browser-side hotkeys service.

See `@kbn/core-hotkeys-browser` for public hotkey types and for **`HotkeysSidebarState` / `HotkeysSidebarActions`** (`openToFeature`, etc.) used with `chrome.sidebar.getApp('hotkeys')`.
