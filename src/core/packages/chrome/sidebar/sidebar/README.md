# @kbn/core-chrome-sidebar

Public types, constants, and validation utilities for the Chrome Sidebar API. Use this package for type imports only; UI and hooks live in `@kbn/core-chrome-sidebar-components`.

This package contains:
- TypeScript types (`SidebarApp`, `SidebarAppDefinition`, `SidebarSetup`, `SidebarStart`, `SidebarComponentProps`, etc.)
- Constants (`VALID_SIDEBAR_APP_IDS`)
- Validation utilities (`isValidSidebarAppId`)

For components and hooks, use `@kbn/core-chrome-sidebar-components`.
