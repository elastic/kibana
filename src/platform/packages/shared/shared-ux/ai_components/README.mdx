## AI components

Shared AI-related visual components for Kibana.

For now, this package contains the `AiButton` components: `AiButtonBase`, `AiButtonAccent`, `AiButtonEmpty`, and `AiButtonIconOnly` (see `ai_button/`). In the future we plan to add other AI-related visual components.

## Usage

```tsx
import { AiButtonBase } from '@kbn/shared-ux-ai-components';

<AiButtonBase onClick={() => undefined}>AI Button</AiButtonBase>;

<AiButtonAccent onClick={() => undefined}>AI Button</AiButtonAccent>;

<AiButtonIcon iconType="aiLogo" onClick={() => undefined} />;

<AiButtonEmpty onClick={() => undefined}>AI Button</AiButtonEmpty>;
```
