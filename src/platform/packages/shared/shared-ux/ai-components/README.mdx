## AI components

Shared AI-related visual components for Kibana.

For now, this package contains the `AiButton` components: `AiButton` (parent), `AiButtonDefault`, `AiButtonAccent`, `AiButtonEmpty`, and `AiButtonIcon` (see `ai_button/`). In the future we plan to add other AI-related visual components.

## Usage

```tsx
import { AiButton, AiButtonDefault } from '@kbn/shared-ux-ai-components';

<AiButton onClick={() => undefined}>AI Button</AiButton>;

<AiButtonDefault onClick={() => undefined}>AI Button</AiButtonDefault>;

<AiButtonAccent onClick={() => undefined}>AI Button</AiButtonAccent>;

<AiButtonIcon iconType="aiAssistantLogo" onClick={() => undefined} />;

<AiButtonEmpty onClick={() => undefined}>AI Button</AiButtonEmpty>;
```
