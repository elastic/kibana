# Navigation extension templates

Navigation extension templates are micro ui elements with a set of defined functionality that created specifically to accommodate interaction with
dynamic user generated data within the navigation bar. The main ethos for the navigation extension templates is that it's behavior can be 
configured by its consumer to suit the experience they would like to craft within the navigation component as javascript serializable records.
