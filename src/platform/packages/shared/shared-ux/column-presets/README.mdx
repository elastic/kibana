## Column presets

This package contains a set of common column presets and utilities for `EuiBasicTable` and `EuiInMemoryTable`
components. Presets are partial configuration objects you pass to the `columns` objects, which automatically
configure properties like `width` and the cell `render()` function with reasonable defaults.

The goal of this package is to unify tabular data rendering across Kibana.
