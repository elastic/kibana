# Feedback Example

Minimal test application for the `@kbn/ui-feedback` package.

## Setup

```bash
# From the Kibana root
yarn kbn bootstrap

# Build the package
cd src/platform/kbn-ui/feedback/packaging
./scripts/build.sh
```

## Run

```bash
cd src/platform/kbn-ui/feedback/packaging/example
yarn start
```

Opens http://localhost:3000.
