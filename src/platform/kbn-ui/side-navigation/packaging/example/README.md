# SideNavigation Example

Minimal test application for the `@kbn/ui-side-navigation` package.

## Setup

```bash
# From the Kibana root
yarn kbn bootstrap

# Build the package
cd src/platform/kbn-ui/side-navigation/packaging
./scripts/build.sh
```

## Run

```bash
cd src/platform/kbn-ui/side-navigation/packaging/example
yarn start
```

Opens http://localhost:3000.
