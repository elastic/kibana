# GridLayout Example

Minimal test application for the `@kbn/ui-grid-layout` package.

## Setup

```bash
# From the Kibana root
yarn kbn bootstrap

# Build the package
cd src/platform/kbn-ui/grid-layout/packaging
./scripts/build.sh
```

## Run

```bash
cd src/platform/kbn-ui/grid-layout/packaging/example
yarn start
```

Opens http://localhost:3000.
