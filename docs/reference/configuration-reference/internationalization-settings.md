# i18n settings in {{kib}} [i18n-settings-kb]


You do not need to configure any settings to run Kibana in English.


## General i18n settings [general-i18n-settings-kb]

`i18n.locale`
:   {{kib}} supports the following locales:

    * English - `en` (default)
    * Chinese - `zh-CN`
    * Japanese - `ja-JP`
    * French - `fr-FR`
