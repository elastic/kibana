# Reporting settings in {{kib}} [reporting-settings-kb]

You can configure `xpack.reporting` settings to:

* [Enable or disable the {{report-features}}](./reporting-settings-enable.md)
* [Configure an encryption key to protect sensitive authentication data](./reporting-settings-encryption-key.md)
* [Choose an access control model of how users will be granted privileges to {{report-features}}](./reporting-settings-security.md)
* [Manage the way reporting tasks run in the {{kib}} server background](./reporting-settings-background-job.md)
* [Control how screenshots are captured for PNG/PDF reports](./reporting-settings-png-pdf.md)
* [Control the limits and capabilities of CSV reports](./reporting-settings-csv.md)

:::{note}
If a setting is applicable to {{ech}} environments, its name is followed by this icon: ![logo cloud](https://doc-icons.s3.us-east-2.amazonaws.com/logo_cloud.svg "Supported on Elastic Cloud Hosted")
:::
