[[auto-formatting]]
== 自动格式化

控制台允许你自动格式复杂的请求，为此,请将光标置于你想格式化的请求上，并从操作菜单中选择自动缩进：

.自动缩进一个请求
image::images/auto_format_before.png["Auto format before",width=500,align="center"]


控制台将调整请求的JSON体，之后请求会像图片所示：

.格式化的请求
image::images/auto_format_after.png["Auto format after",width=500,align="center"]

如果你选择自动缩进一个已经完美格式化后的请求，控制台将把每个文档的请求体折叠到一行，这在使用 Elasticsearch 的批量 API 的时候会非常方便。

.每个文档一行
