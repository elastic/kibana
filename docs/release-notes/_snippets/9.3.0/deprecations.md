## 9.3.0 [kibana-9.3.0-deprecations]

**Data ingestion and Fleet**

::::{dropdown} Show callout for deprecated (in-use) ILM policies.
The `logs`, `metrics`, and `synthetics` ILM policies have been deprecated and replaced with `{name}@lifecycle`. If using a deprecated ILM policy with a component template, the user will see a callout in Fleet > Data Streams with instructions on how to migrate to the new lifecycle policy.

For more information, check [#243416](https://github.com/elastic/kibana/pull/243416) [#6084](https://github.com/elastic/ingest-dev/issues/6084).
