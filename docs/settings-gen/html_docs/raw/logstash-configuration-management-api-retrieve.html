Retrieve pipeline API

edit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will work to fix any issues, but features in technical preview are not subject to the support SLA of official GA features. Retrieve a centrally-managed Logstash pipeline.

Request

edit

GET <kibana host>:<port>/api/logstash/pipeline/<id>

Path parameters

edit
id
(Required, string) The pipeline ID.

Example

edit

The API returns the following:

{
  "id": "hello-world",
  "description": "Just a simple pipeline",
  "username": "elastic",
  "pipeline": "input { stdin {} } output { stdout {} }",
  "settings": {
    "queue.type": "persistent"
  }
}
