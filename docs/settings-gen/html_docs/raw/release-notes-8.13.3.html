The 8.13.3 release includes the following bug fixes.

Alerting

Elastic Security
For the Elastic Security 8.13.3 release information, refer to Elastic Security Solution Release Notes.
Fleet
  • Fixes managed agent policy preconfiguration update (#181624).

  • Fixes KQL/kuery for getting Fleet Server agent count (#180650).
Index Management
  • Fixes allow_auto_create field in the Index Template form (#178321).
Lens & Visualizations
  • Fixes controls on fields with custom label (#180615).
Machine Learning
  • Fixes deep link for Index data visualizer & ES|QL data visualizer (#180389).
Observability
  • Make anomalyDetectorTypes optional (#180717).
SharedUX
  • Revert change to shared UX markdown component for dashboard vis (#180906).
Sharing
  • Default to saved object description when panel description is not provided (#181177).
