Kibana 8.11.2

edit

The 8.11.2 release includes the following bug fixes.

Security updates

edit
  • The 8.11.2 patch release contains a fix for a potential security vulnerability. Please see our security advisory for more details.

Enhancements

edit
APM
  • Added context_propagation_only APM agent setting (#170405).
Elastic Security
For the Elastic Security 8.11.2 release information, refer to Elastic Security Solution Release Notes.
Fleet
  • Improve UX for policy secrets (#171405).
Observability
  • Adds date_formats to SLI ingest pipeline template (#172377).
Platform
  • It is now possible to hot reload Kibana’s TLS (server.ssl) configuration by updating it and then sending a SIGHUP signal to the Kibana process (#171823).

Bug Fixes

edit
Dashboard
  • Fixes reference extract method (#171360).
  • Adds Dashboard title to browser tab title (#171255).
Elastic Security
For the Elastic Security 8.11.2 release information, refer to Elastic Security Solution Release Notes.
Fleet
  • Support integration secrets in a local package registry with variables secret: true and required: false (#172078).
  • Fixes agents metrics retrieval on the agent list page, previously displaying N/A for metrics for users with more than 10 agents. (#172016).
  • Only add time_series_metric if TSDB is enabled (#171712).
  • Fixes inability to upgrade agents from version 8.10.4 to version 8.11 (#170974).
Lens & Visualizations
  • Handle invalid values gracefully for static value operation in Lens (#172198).
  • Make the dashboard SO lighter (#172130).
Machine Learning
  • Fixes blocked jobs polling interval (#171878).
Management
  • Fixes autocomplete to show suggestions even if user types in every letter (#171952).
  • Fixes wrong autocomplete suggestions when using the slash symbol (#171948).
  • Fixes clusters and shards table expanded row not updating when request is changed (#171232).
Maps
  • Fixes using max_result_window to set up a mapbox vector tile (MVT) size request leading to all results not showing (#171344).
Observability
  • Fixes the custom threshold document link (#171125).
SharedUX
  • Fixes custom branding for users without "Saved Object Management" privilege (#171308).
  • Fixes the custom threshold document link (#171125).
Uptime
  • Fixes advanced fields broken for ICMP monitors (#171161). Ê
