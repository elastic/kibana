Dev Tools

edit

Dev Tools contains tools that you can use to interact with your data.

Console

Interact with the REST APIs of Elasticsearch and Kibana, including sending requests and viewing API documentation.

Search Profiler

Inspect and analyze your search queries.

Grok Debugger   

Build and debug grok patterns before you use them in your data processing pipelines.

Painless Lab

[beta] This functionality is in beta and is subject to change. The design and code is less mature than official GA features and is being provided as-is with no warranties. Beta features are not subject to the support SLA of official GA features. Test and debug Painless scripts in real-time.
