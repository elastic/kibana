Delete role API

edit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will work to fix any issues, but features in technical preview are not subject to the support SLA of official GA features. Delete a Kibana role.

Prerequisite

edit

To use the delete role API, you must have the manage_security cluster privilege.

Request

edit

DELETE <kibana host>:<port>/api/security/role/my_admin_role

Response codes

edit
204
Indicates a successful call.
404
Indicates an unsuccessful call.
