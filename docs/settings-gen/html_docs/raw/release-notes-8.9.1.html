Kibana 8.9.1

edit

Review the following information about the Kibana 8.9.1 release.

Breaking changes

edit

Breaking changes can prevent your application from optimal operation and performance. Before you upgrade to 8.9.0, review the breaking changes, then mitigate the impact to your application.

There are no breaking changes in the Kibana 8.9.1 release.

To review the breaking changes in the previous release, check 8.9.0.

Bug fixes

edit
APM
  • Fixes flame graph rendering on the transaction detail page (#162968).
  • Check if documents are missing span.name (#162899).
  • Fixes transaction action menu for Trace Explorer and dependency operations (#162213).
Canvas
  • Fixes embeddables not rendering in Canvas (#163013).
Discover
  • Fixes grid styles to enable better content wrapping (#162325).
  • Fixes search sessions using temporary data views (#161029).
  • Make share links and search session information shorter for temporary data views (#161180).
Elastic Security
For the Elastic Security 8.9.1 release information, refer to Elastic Security Solution Release Notes.
Fleet
  • Fixes for query error on Agents list in the UI (#162816).
  • Remove duplicate path being pushed to package archive (#162724).
Management
  • Resolves potential errors present in v8.9.0 with data views that contain field filters that have been edited (#162860).
Uptime
  • Fixes Monitor not found 404 message display (#163501).

Enhancements

edit
Discover
  • Set legend width to extra large and enable text wrapping in legend labels (#163009).
