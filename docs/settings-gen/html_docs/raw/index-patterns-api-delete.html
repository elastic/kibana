Delete index pattern API

edit

Deprecated in 8.0.0.

Use Delete data view API instead.

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will work to fix any issues, but features in technical preview are not subject to the support SLA of official GA features. Delete Kibana index patterns.

Once you delete an index pattern, it cannot be recovered.

Request

edit

DELETE <kibana host>:<port>/api/index_patterns/index_pattern/<id>

DELETE <kibana host>:<port>/s/<space_id>/api/index_patterns/index_pattern/<id>

Path parameters

edit
space_id
(Optional, string) An identifier for the space. If space_id is not provided in the URL, the default space is used.
id
(Required, string) The ID of the index pattern you want to delete.

Response code

edit
200
Indicates that index pattern is deleted. Returns an empty response body.

Example

edit

Delete an index pattern object with the my-pattern ID:

$ curl -X DELETE api/index_patterns/index_pattern/my-pattern
