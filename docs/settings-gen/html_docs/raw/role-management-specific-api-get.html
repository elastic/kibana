Get specific role API

edit

[preview] This functionality is in technical preview and may be changed or removed in a future release. Elastic will work to fix any issues, but features in technical preview are not subject to the support SLA of official GA features. Retrieve a specific role.

Prerequisite

edit

To use the get specific role API, you must have the manage_security cluster privilege.

Request

edit

GET <kibana host>:<port>/api/security/role/my_restricted_kibana_role

Response code

edit
200
Indicates a successful call.
