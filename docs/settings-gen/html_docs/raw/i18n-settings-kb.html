i18n settings in Kibana

edit

You do not need to configure any settings to run Kibana in English.

General i18n settings

edit
i18n.locale

Kibana supports the following locales:

  • English - en (default)
  • Chinese - zh-CN
  • Japanese - ja-JP
  • French - fr-FR
