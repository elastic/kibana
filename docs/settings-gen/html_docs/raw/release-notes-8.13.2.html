Kibana 8.13.2

edit

The 8.13.2 release includes the following bug fixes.

Bug Fixes

edit
Canvas
  • Fixes text settings to be honored in Canvas markdown elements (#179948).
  • Fixes custom styling for Canvas markdown Workpads (#180070).
Discover
  • Fixes keyboard navigation for search input on the document viewer flyout (#180022).
Elastic Security
For the Elastic Security 8.13.2 release information, refer to Elastic Security Solution Release Notes.
Fleet
  • Fixes having to wait ten minutes after agent upgrade if agent cleared watching state (#179917). Fixes using the latest available version in K8’s manifest instead of the latest compatible version (#179662).
  • Fixes a step in add agent instructions where a query to get all agents was unnecessary (#179603).
Lens & Visualizations
  • Fixes custom styling for TSVB markdown visualizations (#180053).
Machine Learning
  • Single Metric Viewer embeddable: Ensures the detector index is passed to chart correctly (#179761).
  • AIOps: Fixes text field candidate selection for log rate analysis (#179699).
Management
  • Fixes the Response tab loading time to be faster (#180035).
Maps
  • Fixes APM data view ID (#179257).
Monitoring
  • Fixes a runtime error by adding a default value for source and target (#180043).
Operations
  • Fixes an issue with Kibana looking for a configuration file outside of the Kibana home directory, potentially preventing startup due to insufficient permissions (#179847).
