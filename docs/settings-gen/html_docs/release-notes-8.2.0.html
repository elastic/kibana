Kibana 8.2.0

edit

Review the following information about the Kibana 8.2.0 release.

Known issue

edit
Lens & visualizations
A change in the Markdown library that Kibana uses to create TSVB Markdown visualizations and Text dashboard panels renders some tables differently. For more information, check out #767.

Breaking changes

edit

Breaking changes can prevent your application from optimal operation and performance. Before you upgrade, review the breaking changes, then mitigate the impact to your application.

There are no breaking changes in Kibana 8.2.0.

To review the breaking changes in previous versions, refer to the following:

8.1.0 | 8.0.0 | 8.0.0-rc2 | 8.0.0-rc1 | 8.0.0-beta1 | 8.0.0-alpha2 | 8.0.0-alpha1

Features

edit

Kibana 8.2.0 adds the following new and notable features.

Alerting
  • Keep the number_of_scheduled_actions in event log #128438
  • Remove defaultRuleTaskTimeout and set ruleType specific timeout from kibana.yml #128294
  • Limit the executable actions per rule execution #128079 and #126902
Cases
  • Adds Cases to the Stack Management page as a technical preview feature #125224
Dashboard
  • Adds time slider control #128305
  • Adds Control group search settings #128090
  • Adds hierarchical chaining setting to Controls #126649
  • Adds options list API and validation system #123889
Discover
  • Enables document explorer by default #125485
  • Adds Copy to clipboard ability for column name of Document Explorer #123892
Elastic Security
For the Elastic Security 8.2.0 release information, refer to Elastic Security Solution Release Notes.
Lens & Visualizations
  • Adds manual annotations in Lens #126456
  • Adds multi-field group by in TSVB #126015
  • Adds ability to navigate to Lens with your current TSVB configuration #114794
Machine Learning
  • Add link to maps in charts section of Anomaly Explorer #128697
  • Testing trained models in UI #128359
  • Space aware trained models #123487
Management
  • Adds support for auto-complete for data streams #126235
  • Adds ability to filter Data View UI for runtime fields #124114
  • Adds ability to share data views across spaces via data view management #123991
Observability
  • Adds button which allows users to signup for the Synthetics service public beta #128798
  • Adds "View in App URL" {{context.viewInAppUrl}} variable to the rule templating language #128281
  • Adds "View in App URL" {{context.viewInAppUrl}} variable to the rule templating language #128243
  • Adds "View in App URL" {{context.viewInAppUrl}} variable to the rule templating language #127890
  • Adds view in app url as an action variable in the alert message for uptime app #127478

For more information about the features introduced in 8.2.0, refer to What’s new in 8.2.
