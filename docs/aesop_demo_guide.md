### Act 2: Self-Exploration (10-15 min)

**What's happening**: Agent autonomously explores Elasticsearch (read-only), discovers patterns, proposes skills

**Steps**:

1. **Navigate to**: http://localhost:5601/app/evals/aesop/exploration

2. **Configure exploration**:
   - Agent Role: "SOC analyst" (default)
   - Scoped Indices: `.alerts-*`, `.siem-signals-*`, `logs-*` (default)
   - Exploration Depth: 100 samples per index
   - Min Pattern Frequency: 10 instances

3. **Click**: "Start Self-Exploration"

4. **Monitor progress**:
   - Phase 1: Schema Discovery (discovering indices...) ~2 min
   - Phase 2: Data Profiling (sampling documents...) ~3 min
   - Phase 3: Relationship Mapping (testing joins...) ~2 min
   - Phase 4: Pattern Mining (analyzing personas...) ~3 min
   - Phase 5: Skill Synthesis (generating skills...) ~5 min

5. **Workflow completes**:
   ```
   ✅ AESOP Self-Exploration Complete!

   📊 Discovery Summary:
     Indices discovered: 47
     Security indices: 12
     Relationships validated: 8
     Patterns identified: 15
     Skills proposed: 5

   🎯 Next: Review skills at /app/evals/aesop/skills/proposed
   ```

**Behind the scenes** (Workflows YAML execution):
- ✅ All queries are READ-ONLY (verified in audit logs)
- ✅ Agent invocations via `ai.agent` workflow steps
- ✅ Results stored in `.aesop-proposed-skills` index
- ✅ OTEL traces emitted to `traces-*` (viewable in UI)

## Troubleshooting

### Issue: Exploration workflow fails

**Check**:
```bash
# 1. Verify Elasticsearch is running
curl http://localhost:9200

# 2. Check Workflows plugin enabled
curl http://localhost:5601/api/status | jq '.status.plugins.workflowsManagement'

# 3. Check workflow execution logs
# (In Kibana Dev Tools)
GET /.kibana-workflows-executions*/_search
{
  "query": { "term": { "workflow.name": "aesop.self_exploration" } },
  "sort": [{ "@timestamp": "desc" }],
  "size": 1
}
```

---

### Issue: No OTEL traces appearing

**Check**:
```bash
# 1. Verify EDOT collector running
curl http://localhost:4318/v1/traces

# 2. Check telemetry config in kibana.dev.yml
grep -A 5 "telemetry.tracing" config/kibana.dev.yml

# 3. Check traces index exists
curl -u elastic:changeme "http://localhost:9200/_cat/indices/traces-*"

# 4. Query for recent traces
# (In Kibana Dev Tools)
GET /traces-*/_search
{
  "query": { "range": { "@timestamp": { "gte": "now-1h" } } },
  "size": 1
}
```

---

### Issue: Skills not generating

**Check**:
```bash
# 1. Verify custom agents exist
# (In Kibana)
GET /api/agent-builder/agents
# Look for: aesop.schema_categorizer, aesop.pattern_analyzer, aesop.skill_synthesizer

# 2. Check persona behaviors exist
curl -u elastic:changeme "http://localhost:9200/.aesop-persona-behaviors/_count"

# 3. Check workflow step outputs
GET /.kibana-workflows-executions*/_doc/<execution-id>
```
