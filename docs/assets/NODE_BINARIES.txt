### Node binaries information
Here you can find some help to identify the different Node.js distributables.

When building Kibana, we're downloading and bundling our custom versions with glibc 2.17 compatibility,
that are built using this repository: https://github.com/elastic/kibana-custom-nodejs-builds.
This is to keep compatibility for Node.js 18 on older platforms that Kibana supports, but official Node.js builds don't (RHEL7/CentOS7).
