# Project History

This directory contains investigation reports for failed-test tickets assigned to the obs-exploration team in the elastic/kibana repository.

## Format

Each entry is a Markdown file named `YYYY-MM-DD.md` containing:
- Date of investigation
- List of tickets investigated
- Fresh vs Stale status (whether re-investigated or reused from prior run)
- Diagnosis and findings for each ticket
- Recommended actions

## Guidelines

- One file per investigation run (date-stamped)
- Include ticket numbers so future runs can identify previously investigated tickets
- Note when a ticket has been closed or resolved since the last investigation
- Track tickets that persist across multiple runs as potential escalation candidates
